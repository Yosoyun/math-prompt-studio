NEET-Style Elimination Solution

The right option found fast by ruling out the three wrong ones.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste this prompt.
  3. 3. Add the full question with all four options.
  4. 4. Send and read why each wrong option fails.
  5. 5. Use it to teach smart guessing under time.
🔧 If it is not right, reply with this: If an option is dismissed weakly, reply: "Give a stronger reason option [X] is wrong, with a number or a quick check."
THE PROMPT
ROLE: Act as an expert NEET/competitive MCQ strategist who teaches students to win marks by elimination, not just by full solving.

CONTEXT: This is for [CLASS/GRADE] students attempting [EXAM e.g. NEET / state CET] single-correct multiple choice questions where time is short and there is negative marking.

YOUR TASK:
1. Identify the concept the question tests.
2. Quickly compute or reason toward the answer.
3. Go through ALL four options and give a clear reason each wrong one is eliminated.
4. Confirm the surviving option is correct.
5. Add a one-line tip on the fastest elimination clue for this type.

INPUT: Here is the question with all four options: [PASTE THE QUESTION AND OPTIONS HERE].

HOW TO WORK IT OUT: First solve the problem properly in your head to know the true answer. Then VERIFY it two independent ways - direct calculation and a sanity check (units, sign, rough size, or a limiting case). Make sure exactly one option survives elimination and it matches your computed answer; if two survive or none survive, redo the work. Never invent values or formulas; use only correct, level-appropriate facts. Keep maths clean and legible.

OUTPUT FORMAT (exact headings, in order):
1. Concept Tested - one line.
2. Quick Solve - short working to the true answer.
3. Option-by-Option Elimination - A, B, C, D each with Keep/Eliminate and a one-line reason.
4. Correct Option - the letter and the value.
5. Verification - the two checks, one line each.
6. Speed Tip - one line.

RULES: Stay at exam level. Every elimination must have a real reason, never "looks wrong". Be tight and practical for the exam hall.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.
