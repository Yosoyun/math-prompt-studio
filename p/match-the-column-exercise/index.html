Match-the-Column Exercise

A two-column matching exercise (e.g. formula to name, problem to answer) with a clear matching key.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Fill class, board and chapter.
  2. 2. Say what to match (term to definition, etc.).
  3. 3. Choose number of pairs.
  4. 4. Send the prompt.
  5. 5. Use the key to grade quickly.
🔧 If it is not right, reply with this: If one item in Column A matches two in Column B, reply: 'Item 3 in Column A has two possible matches - make every match one-to-one and unique, then re-verify the key.'
THE PROMPT
ROLE: Act as a mathematics teacher who builds match-the-column exercises.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM e.g. CBSE / ICSE / State Board]. The chapter is [CHAPTER NAME]. I want [NUMBER, e.g. 8] matching pairs.

YOUR TASK:
1. Create Column A and Column B with [NUMBER] items each. I want to match: [CHOOSE - e.g. formula to its name / problem to its answer / term to its definition / graph description to function / value to its property].
2. Ensure every Column A item has exactly ONE correct match in Column B (one-to-one, no ambiguity).
3. SHUFFLE Column B so the order does not give away the answers.
4. Optionally add one or two extra 'distractor' items in Column B that match nothing, to raise difficulty (clearly tell me if you do).

INPUT: Here is the sub-topic and the type of items to match: [PASTE DETAILS OR LEAVE BLANK].

HOW TO WORK IT OUT: Build each correct pair, then VERIFY there is exactly one valid match per Column A item by checking each against every Column B option. Re-check any computed values a second way. Do not invent formulas. Keep maths legible.

OUTPUT FORMAT:
SECTION 1 - EXERCISE: Column A (labelled 1, 2, 3...) and Column B (labelled a, b, c...) side by side, shuffled.
SECTION 2 - ANSWER KEY: e.g. 1-c, 2-a... with a one-line reason for each. Note any distractor items.

RULES: One-to-one matches only. No two Column A items sharing an answer. No vague items. Difficulty suited to the class above.


MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.
