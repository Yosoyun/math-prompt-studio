Mixed-Revision DPP Across Many Chapters

A balanced revision sheet that pulls questions from several chapters so nothing is forgotten.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. List every chapter to revise.
  2. 2. Set class and board.
  3. 3. Paste the prompt and send.
  4. 4. Print Section A for students.
  5. 5. Use the chapter tags to spot weak areas.
🔧 If it is not right, reply with this: If one chapter is over-represented, reply: "Re-balance so each listed chapter gets roughly equal questions, and label every question with its chapter."
THE PROMPT
ROLE: Act as a mathematics teacher who designs cumulative revision so earlier chapters are never forgotten.

CONTEXT: I need a MIXED-REVISION DPP that spreads questions across several chapters for [CLASS/GRADE] students preparing for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET].

YOUR TASK:
1. Take my list of chapters and create exactly 12 questions, spread as evenly as possible across them.
2. Tag each question with its chapter name in brackets, e.g. (Quadratics).
3. Mix difficulty: about 4 easy, 5 moderate, 3 challenging.
4. Provide hints, an answer key, and a short 'which chapters to revise more' note based on the spread.

INPUT: Chapters to revise = [LIST YOUR CHAPTERS, e.g. Quadratics, Trigonometry, Probability]. Class = [CLASS/GRADE]. Board/Exam = [BOARD or EXAM]. Time = [TIME e.g. 40 minutes]. If unclear, assume and state it.

HOW TO WORK IT OUT: Solve every question yourself first. Verify each final answer by two independent checks. Keep coverage balanced across the listed chapters. Never invent formulas or values - use only correct mathematics for [CLASS/GRADE]. Keep all maths legible in clean LaTeX or clear plain text.

OUTPUT FORMAT, in this exact order:
1. TITLE BLOCK: 'Mixed-Revision DPP' with class, board/exam, chapters covered, suggested time, and 'Compiled by Indrajeet Yadav'.
2. COVERAGE TABLE: a small table showing how many questions came from each chapter.
3. SECTION A - QUESTIONS: 12 numbered questions, each tagged with its chapter, printable.
4. SECTION B - HINTS: one one-line hint per question.
5. SECTION C - ANSWER KEY: final answers.
6. SECTION D - FULL SOLUTIONS: clean step-by-step working.

RULES: Every listed chapter must appear. No two consecutive questions from the same chapter where avoidable. Tags must be accurate.

OUTPUT FILE FORMAT: present the complete result as a clean, ready-to-use document I can paste straight into Microsoft Word or Google Docs - a clear bold title, section headings, neatly numbered items, comfortable spacing, and simple tables only where helpful (avoid heavy formatting that breaks on paste). It must also print cleanly to PDF on A4 (File > Print > Save as PDF). Keep all mathematics fully readable (fractions as a/b, powers as x^2, roots as sqrt(x)).

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: 'Prepared by Indrajeet Yadav', and put 'Compiled by Indrajeet Yadav' in the title block.
