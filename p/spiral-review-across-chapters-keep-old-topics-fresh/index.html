Spiral Review Across Chapters (Keep Old Topics Fresh)

A mixed worksheet that revisits several past chapters in small doses, with answer key.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. List all chapters covered so far.
  2. 2. Add class and board.
  3. 3. Send the prompt.
  4. 4. Skim the chapter balance.
  5. 5. Print as weekly review.
🔧 If it is not right, reply with this: If one chapter dominates, reply: "Too many questions from Chapter 2. Spread them evenly so each listed chapter gets equal questions, and re-verify answers."
THE PROMPT
ROLE: Act as a mathematics teacher who designs spiral review sheets that keep earlier chapters from being forgotten.

CONTEXT: I teach [CLASS/GRADE] under [BOARD or EXAM e.g. CBSE / ICSE / State Board]. We have finished several chapters and students forget old ones. I want a worksheet that revisits ALL listed chapters in small, frequent doses (spiral revision).

YOUR TASK:
1. Use these chapters and give EQUAL coverage to each: [LIST CHAPTERS HERE].
2. Create 16 questions total, distributed evenly across the chapters.
3. Within each chapter's share, include one easy recall question and one applied question.
4. Shuffle the order so questions from different chapters are interleaved (not grouped), forcing students to switch context.
5. Tag each question with its chapter in brackets, e.g. [Ch: Trigonometry]. Show total marks.

INPUT: Here are the chapters and any weak ones to weight slightly more: [PASTE CHAPTERS AND NOTES HERE].

HOW TO WORK IT OUT: Solve everything internally first. VERIFY each final answer two independent ways. Never invent formulas or values. Keep all maths fully legible in clean LaTeX or clear plain text.

OUTPUT FORMAT (exact headings, in order):
- TITLE BLOCK: 'Spiral Review', Class, Chapters covered, Total Marks, and 'Compiled by Indrajeet Yadav'.
- MIXED QUESTIONS: Q1-Q16, interleaved, each tagged with its chapter and marks.
- CHAPTER BALANCE TABLE: a small table showing how many questions came from each chapter.
- ANSWER KEY: all final answers Q1-Q16 at the end.

RULES: Equal, fair coverage. Questions interleaved, not blocked by chapter. Single correct answers. Answer key must match. Clear tags and simple wording.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: 'Prepared by Indrajeet Yadav', and put 'Compiled by Indrajeet Yadav' in the title block.
