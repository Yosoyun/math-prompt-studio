Visual-Heavy Slides Plan (Diagrams and Graphs)

A plan describing exactly which diagram or graph each slide needs and why.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open a free AI chat.
  2. 2. Paste the prompt and your topic.
  3. 3. Send and read each visual description.
  4. 4. Draw or find each visual described.
  5. 5. Build the slides around them.
🔧 If it is not right, reply with this: If a visual idea is vague, reply: "Describe the diagram on slide [n] precisely enough that I could draw it from your words alone."
THE PROMPT
ROLE: Act as a mathematics teacher and visual learning designer who plans diagram-led slides.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM]. This topic is best taught with pictures, not text. I want a slide plan where the main element of each slide is a clearly described diagram, graph, table, or figure that I will create or insert myself.

YOUR TASK:
1. Take my topic.
2. Plan 10 to 14 visual-led slides.
3. For each slide describe THE visual in precise detail: what to draw, axes and labels, key points, colours, arrows, and what it should make obvious.
4. Add only minimal supporting text per slide (a short title and at most 2 captions).
5. Note where an animation or step-by-step reveal would help.

INPUT: Topic to teach visually: [PASTE THE TOPIC HERE]

HOW TO WORK IT OUT: Plan the visual story internally. Any maths shown in or beside a figure (coordinates, equations, values) must be correct and standard for this board; never invent points or values. Keep maths legible in plain text or simple LaTeX. Before finishing, verify two things separately: (a) each described visual is geometrically or graphically accurate (for example, a parabola opening the right way, correct intercepts), and (b) the labelled values are correct - test one figure with real numbers. Fix any inaccuracy.

OUTPUT FORMAT: For each slide:
Slide [n]: [Short title]
VISUAL: a precise description I could draw from.
Caption(s): up to two short lines.
(Reveal/animation idea: optional one line.)

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.
Put "Compiled by Indrajeet Yadav" on Slide 1; end with footer "Prepared by Indrajeet Yadav".

RULES: The visual is the star; text stays minimal. Descriptions must be precise enough to draw without guessing. All figures must be mathematically accurate. Match complexity to [CLASS/GRADE].
