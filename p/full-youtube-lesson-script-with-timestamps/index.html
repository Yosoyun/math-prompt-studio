Full YouTube Lesson Script With Timestamps

A complete 10-15 minute lesson script broken into timed segments with talking points.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt and your chapter name.
  3. 3. State the target length in minutes.
  4. 4. Record each segment in order.
  5. 5. Use the timestamps as a checklist.
🔧 If it is not right, reply with this: If a segment feels thin, reply: "Expand segment [number] with one more worked example and a common-error warning."
THE PROMPT
ROLE: Act as a senior YouTube lesson scriptwriter for a mathematics teacher's channel.

CONTEXT: This is a full teaching video for [CLASS/GRADE] students studying [CHAPTER/TOPIC] for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. The teacher records on a whiteboard or tablet and wants a structured script with timestamps so the recording stays on track.

YOUR TASK:
1. Plan a lesson of [TARGET LENGTH e.g. 12] minutes.
2. Break it into clear timed segments (intro, concept, worked examples, common mistakes, recap, CTA).
3. For each segment write what the teacher SAYS plus what to WRITE on the board.
4. Include 2 to 3 fully worked examples of increasing difficulty.
5. Add a short common-mistakes section and a final recap.

INPUT: Here is the chapter and level: [PASTE CHAPTER, TOPIC AND CLASS LEVEL HERE].

HOW TO WORK IT OUT: Solve every example completely before writing the script. VERIFY each final answer by two independent methods and state how you checked it. Never invent formulas or numbers; use only standard correct results. Keep all maths in clean LaTeX or clear plain text that reads well aloud and writes cleanly on a board.

OUTPUT FORMAT (a table-style list, in this order):
TITLE LINE: a click-friendly video title.
SEGMENT MAP: each row as [start-end time] | SAY: spoken lines | BOARD: what to write.
Use these segments in order: 00:00 Hook & intro; then Concept; then Example 1; Example 2; Example 3; Common Mistakes; Recap; CTA.
WORKED-EXAMPLE ANSWERS: list each example's final answer in one place for quick reference.
VERIFICATION NOTE: one line per example confirming the two checks.
DESCRIPTION BOX: 3-line video description with 5 hashtags.

RULES: Times must add up to the target length. Spoken lines must sound natural, not robotic. Every example must be correct and verified. No filler. Keep the maths legible.


MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.
Footer: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title block.
