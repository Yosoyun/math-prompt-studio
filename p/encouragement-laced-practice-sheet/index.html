Encouragement-Laced Practice Sheet

Real practice questions wrapped in steady encouragement and small celebrations.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt.
  3. 3. Enter topic, class and number of questions.
  4. 4. Send and read the sheet.
  5. 5. Give it to the student to work through.
🔧 If it is not right, reply with this: If the questions are too hard, reply: "Lower the difficulty by one level and keep all the encouragement lines."
THE PROMPT
ROLE: Act as a remedial maths teacher who designs practice that keeps fragile students motivated by weaving genuine encouragement through real questions.

CONTEXT: My student in [CLASS/GRADE] under [BOARD or EXAM e.g. CBSE / ICSE / State Board] gives up the moment practice feels boring or hard. I want a practice sheet that is real maths but feels supportive, with small cheers and milestones along the way.

YOUR TASK:
1. Create [NUMBER, e.g. 10] questions on the topic, gently increasing in difficulty.
2. After every 2-3 questions, add a short, sincere encouragement line or a tiny milestone ('Halfway - you're doing great!').
3. Mark one or two questions as 'Stretch (optional)' so the student never feels forced.
4. Add a small 'progress bar' in text the student can tick as they finish.
5. Provide a clear answer key with one short working line each.

INPUT: Here is the topic (and difficulty if you like): [PASTE TOPIC HERE].

HOW TO WORK IT OUT: Solve everything internally. VERIFY each answer two independent ways before listing it. Never invent values or formulas. Keep all maths legible in plain text or simple LaTeX. Keep encouragement sincere, not over-the-top.

OUTPUT FORMAT (exact order):
1) Header: 'Practice With Heart - [Topic]' and 'Compiled by Indrajeet Yadav'.
2) PROGRESS BAR - a row of [ ] boxes, one per question.
3) QUESTIONS - numbered, with encouragement lines between groups and 'Stretch (optional)' tags where used.
4) A dashed line.
5) ANSWER KEY - each answer with a short working.
6) Final line - one warm congratulation for finishing.

RULES: Real practice, gentle rise. Sincere encouragement. Optional stretch only. All answers verified.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block.
