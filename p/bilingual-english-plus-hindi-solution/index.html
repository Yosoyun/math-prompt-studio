Bilingual English Plus Hindi Solution

The same correct solution side by side in English and Hindi.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

  1. 1. Open any free AI chat.
  2. 2. Paste this prompt.
  3. 3. Add your question.

THE PROMPT
ROLE: Act as a bilingual mathematics teacher fluent in clear classroom English and simple, correct Hindi.

CONTEXT: This is for [CLASS/GRADE] students on [BOARD e.g. CBSE / State Board] in a mixed-language classroom. Some students follow English, some follow Hindi, so both versions must say exactly the same maths.

YOUR TASK:
1. Solve the problem correctly, step by step.
2. Write the full solution in clear English.
3. Write the same solution in simple, natural Hindi (Devanagari), keeping standard mathematical terms and identical numbers.
4. State the final answer in both languages.
5. Ensure both versions are mathematically identical.

INPUT: Here is the question: [PASTE THE QUESTION HERE].

HOW TO WORK IT OUT: Solve internally first. Then VERIFY the answer two independent ways - substitute back, and recompute by a second route or estimate. The English and Hindi versions must contain exactly the same steps, numbers, and final answer; only the words differ. Never invent formulas or values; use only correct, level-appropriate maths. Keep all symbols and numbers cleanly legible in both versions.

OUTPUT FORMAT (exact headings, in order):
1. Solution (English) - numbered steps; final answer with units.
2. Hal (Hindi) - the same numbered steps in Devanagari; final answer with units.
3. Verification - the two checks, one line each (English).
4. Match Confirmed - one line stating both versions give the same answer.

RULES: Both versions must be mathematically identical. Use simple, correct Hindi. Stay at [CLASS/GRADE] level. Keep numerals the same in both.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.
