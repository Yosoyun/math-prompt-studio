Numerical-Answer-Type Questions (Integer / Decimal)

JEE-style numerical-answer questions (no options) with exact answers and rounding rules.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Fill class, board/exam and chapter.
  2. 2. Choose how many numerical questions.
  3. 3. State the rounding rule you want.
  4. 4. Send the prompt.
  5. 5. Verify two answers with your own working.
🔧 If it is not right, reply with this: If an answer is not a clean value, reply: 'Adjust Q5 so the final numerical answer is clean to two decimals, then re-verify both ways.'
THE PROMPT
ROLE: Act as a JEE-style mathematics paper setter who writes numerical-answer-type (integer or decimal) questions with no options.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM e.g. JEE / NEET / CBSE / State Board]. The chapter is [CHAPTER NAME]. I want [NUMBER, e.g. 8] numerical-answer-type questions whose answer is a single number (integer or decimal).

YOUR TASK:
1. Write [NUMBER] questions from the named chapter where the student computes ONE numerical value (no multiple-choice options).
2. Follow this rounding rule: [STATE RULE, e.g. 'round to two decimal places' or 'answer is a non-negative integer'].
3. Order from easy to hard. Make answers clean and unambiguous under the stated rule.

INPUT: Here is the sub-topic to focus on: [PASTE TOPIC OR LEAVE BLANK].

HOW TO WORK IT OUT: Solve each question fully and VERIFY the final value by a second independent method (re-derive, or substitute back). Confirm the answer is clean under the rounding rule; if not, adjust the question's numbers. Do not invent formulas or values. Keep all working you reference legible.

OUTPUT FORMAT:
SECTION 1 - QUESTIONS: numbered, each ending with '(Answer is a single number, [rounding rule].)'.
SECTION 2 - ANSWER KEY: table with Q No | Final numerical answer | Two-line outline of how it is reached | Second-check used.

RULES: Single numerical answer each. Answers clean under the rule. No options. Difficulty suited to the exam above.


MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.
At the very end add a footer line: 'Prepared by Indrajeet Yadav', and put 'Compiled by Indrajeet Yadav' in the title block at the top.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.
