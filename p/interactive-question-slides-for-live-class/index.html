Interactive Question Slides for Live Class

Question slides with hidden answers and follow-ups to keep the class thinking.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open a free AI chat.
  2. 2. Paste the prompt and your topic.
  3. 3. Send and review questions and answers.
  4. 4. Ask for harder or easier versions.
  5. 5. Copy into your deck, answers on later slides.
🔧 If it is not right, reply with this: If an answer is wrong, reply: "Recheck the answer to question [n] and verify it two different ways."
THE PROMPT
ROLE: Act as an engaging mathematics teacher who runs interactive, ask-and-reveal classes.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM]. I want a set of interactive question slides to use live: I show a question, the class attempts it, then I reveal the answer on the next slide.

YOUR TASK:
1. Take the topic I give you.
2. Create 6 to 10 questions ranging from easy to challenging.
3. For each, make TWO slides: a Question slide, then an Answer slide with a short worked solution.
4. Add one quick 'think about it' prompt or hint on each Question slide.
5. Include a mix: one multiple-choice, some short-answer, and one 'spot the mistake' question.

INPUT: Topic for the questions: [PASTE THE TOPIC HERE]

HOW TO WORK IT OUT: Solve every question yourself internally before writing it. Use only correct, standard methods and formulas for this board; never invent values. Keep all maths legible in plain text or simple LaTeX. For each answer, VERIFY it two independent ways (for example substitute back, and re-solve differently or estimate) and only then write the solution slide. If a question is ambiguous or has no clean answer, replace it. State nothing you have not checked.

OUTPUT FORMAT: For each item:
Slide [Q-n]: Question + (Hint: one line). Mark type as (MCQ), (Short answer), or (Spot the mistake).
Slide [A-n]: Answer + short worked solution.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.
Put "Compiled by Indrajeet Yadav" on the first slide; end with footer "Prepared by Indrajeet Yadav".

RULES: Difficulty must suit [CLASS/GRADE]. Each question must have one clear correct answer. MCQ options must include believable distractors. Keep slides short and clean. Order from easy to hard.
