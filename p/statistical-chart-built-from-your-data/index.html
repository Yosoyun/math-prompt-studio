Statistical Chart Built from Your Data

A correct bar chart, histogram or pie chart from your numbers, fully labelled.

Makes images - needs an image AI  ·  Best tool:  ChatGPT or Gemini (image)

✅ How to use this effectively

  1. 1. Open an image-capable AI chat.
  2. 2. Paste the prompt below.
  3. 3. Paste your data and chart type.
  4. 4. Send and check the values match.
  5. 5. Download the chart.
🔧 If it is not right, reply with this: If a bar height looks wrong, reply: "Re-read the data; the value for [category] is [number]. Recompute all bars and redraw."
THE PROMPT
ROLE: Act as a statistics teacher who turns raw data into a correct, clearly labelled chart.

CONTEXT: I teach [CLASS/GRADE] under [BOARD or EXAM e.g. CBSE / ICSE / State Board]. The chart must read accurately and be classroom-ready.

YOUR TASK:
1. Read my data table exactly as given.
2. Build the chart type I ask for (bar chart, histogram, or pie chart).
3. Label the axes (or sectors), give the chart a title, and show each value or percentage on the chart.
4. Use evenly spaced, clearly scaled axes; for a pie chart, show each sector's percentage and angle.
5. Keep it clean: distinct but plain colours, readable labels, white background.

INPUT: Here is my data: [PASTE THE DATA TABLE HERE, e.g. Subject : Marks  ->  Maths 80, Physics 65, Chemistry 70]. Chart type I want: [bar / histogram / pie]. Title (optional): [TITLE].

HOW TO WORK IT OUT: First copy my numbers into a small clean table and show it back to me so I can confirm nothing was misread. For a pie chart, compute each sector angle as (value / total) x 360 and show the arithmetic. VERIFY two ways: confirm the bar heights or sector sizes are in the same order and ratio as the data, and confirm pie angles add up to 360 (or percentages to 100). Never round in a way that breaks the total; never invent or drop a category.

OUTPUT FORMAT:
1. DATA AS READ: the table exactly as I gave it.
2. CALCULATIONS: totals, percentages, and pie angles if relevant.
3. THE CHART: one clean labelled image with title, axis/sector labels and values shown.
4. CHECK: one line confirming totals add up correctly.

RULES: Every data value must be represented faithfully. Axes evenly scaled and labelled. Pie angles must sum to 360. No 3D or decorative effects. Plain white background.

SIGNATURE: Sign the bottom-right of every page with an elegant handwritten signature "Indrajeet Yadav" with a small underline flourish.
