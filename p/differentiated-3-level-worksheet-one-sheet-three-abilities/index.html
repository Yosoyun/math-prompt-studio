Differentiated 3-Level Worksheet (One Sheet, Three Abilities)

One topic, three matched difficulty levels (Support/Core/Extension) on the same skills, with key.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Name the topic to differentiate.
  2. 2. Add class and board.
  3. 3. Send the prompt.
  4. 4. Check all three levels match.
  5. 5. Print the right level per group.
🔧 If it is not right, reply with this: If the levels overlap too much, reply: "Make the gap between Support, Core, and Extension clearer while testing the same skills, and re-verify all keys."
THE PROMPT
ROLE: Act as a mathematics teacher who designs differentiated worksheets for mixed-ability classes.

CONTEXT: I teach [CLASS/GRADE] under [BOARD or EXAM e.g. CBSE / ICSE / State Board]. My class has weak, average, and strong students in the same room. I want ONE topic, [TOPIC], delivered as three parallel worksheets at three levels, all testing the SAME underlying skills so I can teach together but assign by ability.

YOUR TASK:
1. Produce three versions of a worksheet on [TOPIC], each with 8 questions covering the same skills in the same order:
 - LEVEL 1 SUPPORT: easiest, small friendly numbers, more guidance, sometimes a hint in the question.
 - LEVEL 2 CORE: standard difficulty, the expected level for this class.
 - LEVEL 3 EXTENSION: harder, larger or trickier numbers, an extra reasoning twist on each.
2. Keep question 1 across all three levels about the same skill (just harder versions), and so on for questions 2-8, so the class stays in sync.
3. Show marks and totals for each level.

INPUT: Here is the topic and the core skills it must test: [PASTE TOPIC AND SKILLS HERE].

HOW TO WORK IT OUT: Solve every question in all three levels internally first. VERIFY each final answer two independent ways. Confirm that question N tests the same skill across all three levels, only the difficulty differs. Never invent formulas or values. Keep all maths fully legible in clean LaTeX or clear plain text.

OUTPUT FORMAT (exact headings, in order):
- TITLE BLOCK: 'Differentiated Worksheet - 3 Levels', Topic, Class, and 'Compiled by Indrajeet Yadav'.
- LEVEL 1 SUPPORT: 8 questions with marks and total.
- LEVEL 2 CORE: 8 questions with marks and total.
- LEVEL 3 EXTENSION: 8 questions with marks and total.
- ANSWER KEY: three short blocks (Support, Core, Extension), final answers Q1-Q8 each, at the end.

RULES: Same skill per question number across levels; clear difficulty gaps. Single correct answers. Each key must match its level. Simple wording.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.
