Printable A4 Worksheet Layout (Clean Handout)

A neatly laid-out, print-ready A4 worksheet with name/date line, instructions, and key.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Name the topic and question count.
  2. 2. Add class and board.
  3. 3. Send the prompt.
  4. 4. Copy text into Word on A4.
  5. 5. Print and photocopy.
🔧 If it is not right, reply with this: If it overflows a page, reply: "This is too long for one A4 page. Reduce to fit one page neatly while keeping the name line and answer key."
THE PROMPT
ROLE: Act as a mathematics teacher who formats clean, print-ready A4 worksheets.

CONTEXT: I teach [CLASS/GRADE] under [BOARD or EXAM e.g. CBSE / ICSE / State Board]. I will copy your output into a document and print it on A4 paper to photocopy for the class. So the layout must be tidy, well spaced, and printer-friendly (no fancy graphics, just clean text and structure).

YOUR TASK:
1. Create a worksheet on [TOPIC] with [NUMBER, e.g. 12] questions, ordered easy to hard, with marks and a total.
2. Lay it out clearly for A4: a header block, a 'Name: ______ Class: ____ Date: ____' line, a one-line instruction, then the questions with enough spacing implied (use clear numbering and line breaks).
3. Keep line length reasonable so it does not wrap badly when pasted into Word.
4. Put the answer key on a clearly separated final section labelled 'TEACHER COPY - ANSWER KEY' so I can print it separately if I want.
5. Avoid characters that break in Word; use plain text maths or simple LaTeX I can read.

INPUT: Here is the topic and number of questions: [PASTE TOPIC AND COUNT HERE].

HOW TO WORK IT OUT: Solve every question internally first. VERIFY each final answer two independent ways. Never invent formulas or values. Keep all maths fully legible in clean plain text or simple LaTeX, and keep formatting copy-paste safe.

OUTPUT FORMAT (exact headings, in order):
- HEADER BLOCK: 'Worksheet: [Topic]', Class, Total Marks, Time, and 'Compiled by Indrajeet Yadav'.
- STUDENT INFO LINE: Name / Class / Date blanks.
- INSTRUCTIONS: one short line.
- QUESTIONS: Q1 to Q[NUMBER] with marks, well spaced.
- TEACHER COPY - ANSWER KEY: all final answers, clearly separated at the end.

RULES: Tidy, printer-friendly, single page if reasonable. Copy-paste safe characters. Single correct answers. Key must match. Simple wording.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: 'Prepared by Indrajeet Yadav', and put 'Compiled by Indrajeet Yadav' in the header block.
