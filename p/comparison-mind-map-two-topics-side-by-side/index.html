Comparison Mind Map (Two Topics Side by Side)

A map contrasting two related topics: same, different, and when to use each.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Name the two topics to compare.
  3. 3. Paste the prompt below.
  4. 4. Send and study the differences.
  5. 5. Ask for a quick rule to tell them apart.
🔧 If it is not right, reply with this: If a difference is vague, reply: "Make that difference concrete with a one-line example for each topic." If something is wrong, reply: "Re-check that row and verify both sides."
THE PROMPT
ROLE: Act as a mathematics teacher who builds comparison mind maps that contrast two related topics students confuse.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. Students mix up two related topics. I want a COMPARISON MIND MAP (text form) that puts the two side by side - what they share, where they differ, and exactly when to use each. No image needed.

YOUR TASK:
1. Name both topics as the two main branches.
2. Create a SHARED branch for what they have in common.
3. Create a DIFFERENCES section comparing them point by point (definition, formula, method, conditions, typical use).
4. Add a DECISION RULE: how to instantly tell which one a question needs.
5. Add one short example of each that shows the contrast clearly.

INPUT: Here are the two topics to compare: [PASTE TOPIC A AND TOPIC B HERE]. If I give only names, use the standard [BOARD] meaning for [CLASS/GRADE].

HOW TO WORK IT OUT: First list the true similarities and real differences privately. VERIFY each comparison point twice - is it accurate for both topics, and is the distinction real? Never invent a difference; if two things are actually the same, say so. Mark doubt as "(verify)". Keep all maths in clean plain text.

OUTPUT FORMAT (exactly this):
COMPARISON: <Topic A> vs <Topic B>
SHARED (what they have in common):
   - point
   - point
DIFFERENCES (point by point):
   Aspect: Definition  | <Topic A>: ...  | <Topic B>: ...
   Aspect: Formula     | <Topic A>: ...  | <Topic B>: ...
   Aspect: Method      | <Topic A>: ...  | <Topic B>: ...
   Aspect: Conditions  | <Topic A>: ...  | <Topic B>: ...
   Aspect: Typical use | <Topic A>: ...  | <Topic B>: ...
DECISION RULE: in one line, how to tell which topic a question needs.
CONTRAST EXAMPLES: one mini example of each.

RULES: Differences must be real and concrete. No vague contrasts. Match depth to [CLASS/GRADE].

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block.
