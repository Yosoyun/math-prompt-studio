Homework and Next-Class Link Plan

Aligned homework plus a plan to review it and bridge into the next lesson.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt with both topics.
  3. 3. Send and read the homework set.
  4. 4. Note the next-class review plan.
  5. 5. Print and assign the homework.
🔧 If it is not right, reply with this: If homework is too long, reply: "Cut the homework to 6 questions that take under 25 minutes, keeping the bridge to the next lesson."
THE PROMPT
ROLE: Act as a mathematics teacher who designs homework that connects today's lesson to tomorrow's.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM]. I want homework after today's lesson that reinforces what was taught and quietly prepares students for the next lesson, plus a quick plan for how to review the homework at the start of next class. Homework should take about [MINUTES] minutes.

YOUR TASK:
1. Create a graded homework set on today's topic: a few easy, a few medium, one or two that stretch toward the next topic.
2. Provide a full answer key so I can mark quickly.
3. Add a short marking guide: what to look for and common errors.
4. Write a 5-minute next-class review plan that uses the homework as the bridge into the new lesson.
5. Include one 'connector' question that visibly links today's topic to the next one.

INPUT: Today's topic: [PASTE TODAY'S TOPIC HERE]. Next lesson's topic: [PASTE NEXT TOPIC HERE].

HOW TO WORK IT OUT: Solve every homework question internally first. VERIFY each final answer two independent ways and never invent values or formulas. Keep maths legible in clean plain text or simple LaTeX. Estimate the homework time and keep it near the stated minutes.

OUTPUT FORMAT (exact headings, in order):

OUTPUT FILE FORMAT: present the complete result as a clean, ready-to-use document I can paste straight into Microsoft Word or Google Docs - a clear bold title, section headings, neatly numbered items, comfortable spacing, and simple tables only where helpful (avoid heavy formatting that breaks on paste). It must also print cleanly to PDF on A4 (File > Print > Save as PDF). Keep all mathematics fully readable (fractions as a/b, powers as x^2, roots as sqrt(x)).

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.
1. TITLE BLOCK - today's topic, next topic, class, board, 'Compiled by Indrajeet Yadav'.
2. HOMEWORK SET - graded easy to stretch, clearly numbered.
3. ANSWER KEY - full solutions with the two checks on the harder items.
4. MARKING GUIDE - what to look for and common errors.
5. CONNECTOR QUESTION - the one that bridges to the next topic, with answer.
6. NEXT-CLASS REVIEW PLAN - a 5-minute opener that leads into the new lesson.

RULES: Homework must fit the time limit. The bridge to the next topic must be clear. Every answer verified.

At the very end add a footer line: 'Prepared by Indrajeet Yadav', and put 'Compiled by Indrajeet Yadav' in the title block.
