Graded Exercise Set With Full Solutions

A leveled question set plus a separate fully-worked solutions section.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Enter topic, class, board and question count.
  2. 2. Send the prompt.
  3. 3. Spot-check two solutions for accuracy.
  4. 4. Ask for harder items if too easy.
🔧 If it is not right, reply with this: If an answer is wrong, reply: "Re-solve Question 7 fully, verify the answer by substitution or a second method, and correct the answer key."
THE PROMPT
ROLE: Act as a mathematics question-bank author and examiner.

CONTEXT: I am building a graded exercise for a textbook chapter on [TOPIC] for [CLASS/GRADE] students of [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. I want students to practise from easy to hard, with complete solutions they can learn from.

YOUR TASK:
1. Create [TOTAL e.g. 20] original questions on the topic.
2. Split them into three levels: Level 1 Basic, Level 2 Moderate, Level 3 Challenging.
3. Make difficulty rise smoothly within and across levels.
4. Cover all the main sub-topics and question types of the chapter.
5. Provide full step-by-step solutions in a separate section, not mixed with the questions.

INPUT: Here is the scope and any constraints (sub-topics, marks per question, calculator allowed or not): [PASTE HERE OR WRITE "standard scope"].

HOW TO WORK IT OUT: Solve every question yourself before listing it; never include a question you cannot solve. For each final answer, VERIFY it by two independent checks (for example substitute back AND solve by an alternate route or estimate). Never invent formulas or values. Keep all mathematics in clean LaTeX or clear plain text, one step per line.

OUTPUT FORMAT (exact order):
1. Heading: "Exercise" then the three levels with numbered questions only
2. Page break line: "--- SOLUTIONS ---"
3. Full Solutions (matching question numbers, with steps and a final boxed answer)
4. Quick Answer Key (number and final answer only)

RULES: Questions and solutions must be in separate sections. Every solution must show all steps. Difficulty must be genuinely graded. No unsolved or impossible questions.

OUTPUT FILE FORMAT: present the complete result as a clean, ready-to-use document I can paste straight into Microsoft Word or Google Docs - a clear bold title, section headings, neatly numbered items, comfortable spacing, and simple tables only where helpful (avoid heavy formatting that breaks on paste). It must also print cleanly to PDF on A4 (File > Print > Save as PDF). Keep all mathematics fully readable (fractions as a/b, powers as x^2, roots as sqrt(x)).

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the header block at the top.
