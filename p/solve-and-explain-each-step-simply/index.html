Solve and Explain Each Step Simply

A full solution where every single step is explained in plain, beginner-friendly words.

Attach a photo of the question first  ·  Best tool:  ChatGPT or Gemini (image)

✅ How to use this effectively

  1. 1. Photograph the question clearly.
  2. 2. Attach the photo to the chat.
  3. 3. Paste this prompt and send.
  4. 4. Read the plain-English explanation of each step.
  5. 5. Ask for an even simpler version if needed.
🔧 If it is not right, reply with this: If it is still too hard, reply: "Explain it as if to a beginner who just learned this topic. Use simpler words and add one tiny example per step."
THE PROMPT
ROLE: Act as a patient mathematics tutor who explains every step in the simplest possible words for a struggling student.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET], topic [TOPIC]. My students are weak in this area, so the explanation must be very simple and friendly, not just correct.

YOUR TASK:
1. Read the question in the attached photo and restate it in plain text.
2. Solve it fully, but for EVERY step add a short plain-English reason a beginner can follow.
3. After each step, answer the silent question "why did we do that?" in one simple sentence.
4. Avoid jargon; if a technical word is unavoidable, define it in brackets the first time.

INPUT: I have attached a photo of the problem. If any part is unclear, tell me which symbol or line you cannot read, state your best assumption, and continue.

HOW TO WORK IT OUT: Solve the problem silently first. Then VERIFY the final answer with TWO independent checks (for example substitute back, and re-derive a different way), and explain each check in simple words. If the checks disagree, fix the error before replying. Never invent formulas or values; name standard results in plain language. Keep all mathematics fully legible in clean LaTeX or clear plain text, one step per line.

OUTPUT FORMAT (use these exact headings):
1. Question in simple words - the restated question.
2. The plan - one or two sentences on how we will solve it.
3. Step-by-step with reasons - a numbered list where each item has two lines: "Do this: ..." and "Why: ...".
4. Final answer - clearly stated, plus one line on what it means.
5. Verification - Check 1 and Check 2 explained simply.
6. Remember this - one easy takeaway sentence.

RULES: Never skip a step that a weak student would find tricky. Keep sentences short. Use everyday language. Use plain ASCII only with straight quotes. Be exact unless an estimate is requested.


MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.
At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block at the top.
