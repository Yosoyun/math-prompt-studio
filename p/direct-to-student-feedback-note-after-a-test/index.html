Direct-to-Student Feedback Note After a Test

Honest, motivating feedback written straight to the student.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt; add score and errors.
  3. 3. Note 1 thing the student did well.
  4. 4. Read and tweak the tone.
  5. 5. Hand it to the student or send it.
🔧 If it is not right, reply with this: If it feels harsh or vague, reply: "Keep it motivating, name the exact mistakes, and give one clear next action."
THE PROMPT
ROLE: Act as an encouraging maths teacher writing feedback directly TO a student after a test.

CONTEXT: My student [STUDENT NAME], in [CLASS/GRADE], studying [SUBJECT] for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET], just got a test back. I want feedback that is honest about mistakes but motivating, so the student feels capable, not defeated. The note is addressed to the student (you-voice).

YOUR TASK:
1. Write a short feedback note that names 1 to 2 specific strengths in their work.
2. Point out the main types of mistakes in a constructive way (not a list of every error).
3. Give ONE clear, concrete next action for improvement before the next test.
4. End with a genuine, motivating line that builds belief.

INPUT - the real results (use only these facts):
- Score/marks: [e.g. 14/25]
- Main mistake types: [e.g. sign errors / skipped steps / formula wrong / time ran out]
- One thing done well: [STRENGTH]
- Tone: [warm and motivating / firm but kind]

HOW TO WORK IT OUT: Use only the facts above; never invent marks or mistakes. Focus feedback on the work and method, not the child's character. Keep the next action small and achievable. Re-read and remove anything discouraging or vague.

OUTPUT FORMAT (use these exact headings):
WHAT YOU DID WELL: 1 to 2 specific points.
WHERE MARKS SLIPPED: 1 to 2 mistake patterns, constructive.
YOUR ONE NEXT STEP: a single clear action.
A WORD OF ENCOURAGEMENT: 1 motivating line.

RULES:
- Specific and honest, yet confidence-building.
- Address the student directly; age-appropriate.
- Plain, clean text only.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.

SIGNATURE: End your ENTIRE response with exactly this footer block (keep the lines and symbols):
