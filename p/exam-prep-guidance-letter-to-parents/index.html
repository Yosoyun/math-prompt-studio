Exam-Prep Guidance Letter to Parents

A calm parent guide on how to support their child before exams.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt with exam details.
  3. 3. Add the chapters or syllabus.
  4. 4. Send to the class parents.
  5. 5. Ask for a one-page PDF-style layout if needed.
🔧 If it is not right, reply with this: If advice feels generic, reply: "Tie each tip to my specific chapters and exam date. Remove anything that fits any subject."
THE PROMPT
ROLE: Act as a reassuring maths teacher guiding parents on supporting their child before exams.

CONTEXT: My students in [CLASS/GRADE] studying [SUBJECT] for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET] have exams coming. Many parents want to help but do not know how, and some accidentally add pressure. I want a calm, practical guidance message for parents.

YOUR TASK:
1. Write a short letter explaining how parents can help WITHOUT increasing stress.
2. Give 4 to 6 practical, low-effort tips (study routine, sleep, revision over re-reading, no comparison, healthy food, a quiet study spot).
3. List the key chapters/topics to focus on, taken only from what I provide.
4. End by reminding parents that effort and calm matter more than fear.

INPUT - the real details (use only these facts):
- Exam name and dates: [DETAILS]
- Key chapters/topics: [LIST]
- Anything I want parents to avoid: [e.g. last-minute new topics / comparing with siblings, optional]
- Tone: [calm and warm / formal]

HOW TO WORK IT OUT: Use only the facts above; never invent dates or chapters. Keep advice realistic for a busy Indian household. Avoid fear-based language. Re-read and ensure every tip is concrete and doable, not vague.

OUTPUT FORMAT (use these exact headings):
SUBJECT LINE: calm and supportive.
OPENING: 2 reassuring lines.
HOW YOU CAN HELP AT HOME: 4 to 6 numbered tips.
WHAT TO FOCUS ON: the chapter/topic list.
A GENTLE REMINDER: 1 to 2 closing lines about calm and effort.

RULES:
- Supportive, never pressuring; no scare tactics.
- Practical for ordinary families; no jargon.
- Plain, clean text only.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the header block above the subject line.
