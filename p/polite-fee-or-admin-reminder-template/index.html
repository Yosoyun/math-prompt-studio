Polite Fee or Admin Reminder Template

A respectful, non-awkward reminder for fees, forms, or pending items.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt; name what is due.
  3. 3. Add amount or item and date.
  4. 4. Choose first reminder or follow-up.
  5. 5. Send privately, not in a group.
🔧 If it is not right, reply with this: If it sounds pushy or cold, reply: "Make it warmer and assume good faith - they may have simply forgotten."
THE PROMPT
ROLE: Act as a polite, respectful teacher or tutor sending a sensitive reminder about a pending fee or admin item.

CONTEXT: I need to remind a parent about something pending for their child in [CLASS/GRADE] (e.g. tuition fee, form, document, materials). Money and admin reminders feel awkward, so the tone must be courteous, private, and assume good faith - never embarrassing or accusatory.

YOUR TASK:
1. Write a private, respectful reminder that clearly states what is pending and the due date.
2. Assume the parent simply forgot; keep dignity intact.
3. Offer an easy way to act (payment method, where to submit) and an option to ask if there is an issue.
4. Provide TWO versions: a FIRST gentle reminder and a slightly firmer POLITE FOLLOW-UP.

INPUT - the real details (use only these facts):
- What is pending: [e.g. monthly fee / consent form / ID copy / book payment]
- Amount or item: [DETAILS, optional]
- Original due date and new due date: [DETAILS]
- How to complete it: [UPI / cash in office / submit to me / link]
- Tone: [very gentle / gentle but clear]

HOW TO WORK IT OUT: Use only the facts above; never invent amounts or dates. Avoid any wording that shames or pressures. Keep it short and private (not for a group). Re-read and ensure it stays warm even in the follow-up version.

OUTPUT FORMAT (use these exact headings):
FIRST REMINDER: 60 to 100 words, warm and easy.
POLITE FOLLOW-UP: 60 to 100 words, still courteous, slightly clearer on the date.
ONE-LINE NUDGE: a single friendly line for a quick message.

RULES:
- Respectful and private; assume good faith; never embarrass.
- Clear on item and date; offer help if there is difficulty.
- Plain, clean text only.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the header block at the top.
