Confidence-Building Easy Win Set

A short set of deliberately achievable questions so the student tastes success early.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt.
  3. 3. Enter the topic and class.
  4. 4. Send and read the set.
  5. 5. Give it to the student first thing.
🔧 If it is not right, reply with this: If anything is too hard, reply: "Make question 4 onward easier too - I want the student to get every single one right."
THE PROMPT
ROLE: Act as a remedial maths coach whose only goal here is to rebuild a discouraged student's confidence with easy wins.

CONTEXT: I have a student in [CLASS/GRADE] following [BOARD or EXAM e.g. CBSE / ICSE / State Board] who believes they are 'bad at maths'. I want a short practice set designed so they get almost everything right and feel proud. Difficulty must stay genuinely easy but still real practice on the topic.

YOUR TASK:
1. Create exactly 10 short questions on the topic, all deliberately easy and quick.
2. Start with the easiest possible (almost trivial) and rise only very gently.
3. Use small, round, friendly numbers.
4. Add one short cheerful line of encouragement after every 3 questions.
5. Provide a clear answer key with a one-line working for each.

INPUT: Here is the topic (and any number range I prefer): [PASTE TOPIC HERE].

HOW TO WORK IT OUT: Solve all questions internally. VERIFY every answer two independent ways before listing it (e.g. forward check and reverse check). Never invent formulas or values. Keep maths clean and legible in plain text or simple LaTeX.

OUTPUT FORMAT (exact order):
1) Header: 'You CAN Do This - Easy Wins on [Topic]', with 'Compiled by Indrajeet Yadav'.
2) THE 10 QUESTIONS, numbered, with encouragement lines after Q3, Q6 and Q9.
3) A dashed line.
4) ANSWER KEY: each answer with a single short working line.
5) One closing sentence praising the student for finishing.

RULES: Truly easy. Friendly numbers. Real (not fake) practice. Warm tone. Every answer verified and correct.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.
