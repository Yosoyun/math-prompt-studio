Reward-Chart System for Maths Habits

A star-chart system tied to 5 habits, milestones, a class-wide goal, a worked example and fairness rules.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. Type your class, board and number of weeks.
  2. Paste the prompt and send.
  3. Draw the weekly star chart.
  4. Award stars against the listed habits.
  5. Track milestones and the class goal.
🔧 If it is not right, reply with this: If a milestone is reached too fast or too slowly, reply: "Re-do the star arithmetic so a typical student hits the first milestone in about two weeks, and keep the class goal achievable but not trivial."
THE PROMPT
ROLE: Act as a primary/middle-school mathematics teacher and behaviour-design expert who builds a printable reward-chart system tied to maths habits.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM e.g. CBSE / ICSE / State Board]. I want a reward-chart and sticker/star system that builds good maths HABITS (daily practice, showing working, correcting mistakes) over a [NUMBER]-week period.

YOUR TASK:
1. Define 5 maths habits worth rewarding and how many stars each earns.
2. Design a weekly star chart layout (rows for students or habits, columns for days) I can draw or print.
3. Set star-to-reward milestones (for example 10 stars = a privilege) that cost little or nothing.
4. Add a class-wide goal so the whole group wins together sometimes, not only individuals.
5. Give 3 rules to keep stars fair and meaningful (no inflation).

INPUT: Class, board and number of weeks are above. If I add nothing, assume about 40 students and a tiny budget.

HOW TO WORK IT OUT: Internally check the star arithmetic: confirm how many stars a typical student earns per week, when milestones are realistically reached, and that the class goal is achievable but not trivial. State the totals you used. Never invent unrealistic numbers. Keep all arithmetic in clear plain text.

OUTPUT FORMAT (use these exact headings):
SYSTEM OVERVIEW (2-3 lines)
HABITS AND STAR VALUES (table)
WEEKLY STAR CHART (describe the grid to draw/print)
REWARD MILESTONES (stars needed and the low-cost reward)
CLASS-WIDE GOAL (how the group wins together)
WORKED EXAMPLE (one student's week with star total)
FAIRNESS RULES (3 bullets)
KICK-OFF NOTE (short paragraph to read to the class)

RULES: Star maths must add up and milestones must be realistically reachable. Rewards must be low or no cost. No habit may be impossible to verify quickly. Keep the chart drawable by hand.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.
