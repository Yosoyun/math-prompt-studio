Quick Class Test (20 Marks, 30 Minutes)

A short 20-mark class test that fits in one short period, with a quick answer key.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Paste the prompt into a chat.
  2. 2. Type the topic or chapter.
  3. 3. Set the class level.
  4. 4. Send and skim the short test.
  5. 5. Keep the key handy for marking.
  6. For a trustworthy answer key: paste the finished paper into a FRESH chat and ask it to solve and grade it as an unseen exam (it catches its own slips).
🔧 If it is not right, reply with this: If it is too long for 30 minutes, reply: 'This is too long to finish in 30 minutes. Cut it down to fit comfortably, keep 20 marks, and re-verify the key.'
THE PROMPT
ROLE: Act as a mathematics teacher who writes brisk, fair class tests.

CONTEXT: I need a SHORT class test worth exactly 20 marks that students can finish in about 30 minutes, for Class [CLASS/GRADE], [BOARD or EXAM e.g. CBSE / ICSE / State Board], on the topic: [TOPIC or CHAPTER].

YOUR TASK:
1. Write a small set of questions totalling exactly 20 marks (for example a few one-mark MCQs, some two-mark short questions, and one or two longer questions).
2. Make sure the whole test is genuinely doable in 30 minutes.
3. Keep difficulty mostly easy to moderate with one slightly harder question.
4. Provide a quick answer key.

INPUT: The topic is: [TOPIC or CHAPTER]. Use only this topic.

HOW TO WORK IT OUT: Solve every question internally, then verify each final answer twice by independent checks. Estimate the time each question takes and confirm the total is around 30 minutes. Never invent formulas or values. Keep mathematics legible in clean LaTeX or plain text. Confirm marks total exactly 20.

OUTPUT FORMAT, in this order:
1. HEADER: 'Class Test - [TOPIC or CHAPTER]', 'Class: [CLASS/GRADE]', 'Time: 30 minutes', 'Maximum Marks: 20', 'Compiled by Indrajeet Yadav'.
2. INSTRUCTIONS: 2 to 3 short numbered lines.
3. THE TEST: numbered questions, marks beside each.
4. ANSWER KEY: final answer for every question.

RULES: Exactly 20 marks. Finishable in 30 minutes. Stay on the named topic. Number every question.

OUTPUT FILE FORMAT: present the complete result as a clean, ready-to-use document I can paste straight into Microsoft Word or Google Docs - a clear bold title, section headings, neatly numbered items, comfortable spacing, and simple tables only where helpful (avoid heavy formatting that breaks on paste). It must also print cleanly to PDF on A4 (File > Print > Save as PDF). Keep all mathematics fully readable (fractions as a/b, powers as x^2, roots as sqrt(x)).

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: 'Prepared by Indrajeet Yadav', and put 'Compiled by Indrajeet Yadav' in the header.
