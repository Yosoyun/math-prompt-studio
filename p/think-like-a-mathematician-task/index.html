Think Like a Mathematician Task

A task that models the real thinking process, not just the polished answer.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Paste the prompt and the problem.
  2. 2. Set the [CLASS/GRADE] level.
  3. 3. Send and read the thinking narrative.
  4. 4. Discuss the false starts with students.
🔧 If it is not right, reply with this: If it jumps straight to the answer, reply: "Show the messy thinking first - include at least one false start and how you recovered, then verify the final answer twice."
THE PROMPT
ROLE: Act as an expert mathematician who narrates the REAL thinking process - including dead ends - so students learn how problem-solving actually feels.

CONTEXT: My gifted students in [CLASS/GRADE] (level [BOARD or EXAM e.g. RMO / JEE]) think 'real mathematicians just see the answer'. I want a task that shows the honest, messy process: exploring, getting stuck, trying things, recovering, then polishing.

YOUR TASK:
1. Take the problem I give (or pick one on [TOPIC]).
2. Write a 'THINKING OUT LOUD' narrative that models the process: first reactions, what to try first and why, at least one FALSE START and how you noticed it failed, the moment of insight, and how you recovered.
3. Then write the CLEAN final solution as it would appear in a textbook.
4. Reflect on the difference between the messy process and the clean writeup, and name 3 'mathematician habits' the narrative demonstrates (e.g. try small cases, seek symmetry, question assumptions).
5. Give 2 problems for students to practise the same habits.

INPUT: Here is the problem (or topic): [PASTE THE PROBLEM OR TOPIC HERE].

HOW TO WORK IT OUT: Genuinely solve the problem and let the narrative reflect plausible real reasoning, including a believable wrong turn that you then fix. VERIFY the final answer by two independent methods (e.g. an alternate approach plus a numerical or small-case check). The false start must be instructive, not nonsense, and you must clearly show why it failed. Never invent formulas or fake the recovery. Keep all mathematics fully legible in clean LaTeX or clear plain text.

OUTPUT FORMAT (exact headings, in order):
- THE PROBLEM (stated cleanly)
- THINKING OUT LOUD (the messy real process, with a false start and recovery)
- THE CLEAN SOLUTION (polished, textbook-style)
- FINAL ANSWER (clearly boxed)
- VERIFICATION (two independent checks shown)
- MESSY vs CLEAN (what the writeup hides)
- THREE MATHEMATICIAN HABITS ON SHOW
- PRACTISE THESE HABITS (2 problems)

RULES: The false start must be honest and instructive. Final answer double-checked. No bluffing in either the messy or clean version.


MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.
At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the header block.
