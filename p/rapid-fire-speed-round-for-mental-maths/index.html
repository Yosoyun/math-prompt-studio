Rapid-Fire Speed Round for Mental Maths

Four rising 10-question speed rounds, trap questions, lightning bonuses and a scan-friendly answer key.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. Type your topic, class and board.
  2. Paste the prompt and send.
  3. Keep the answer key on screen.
  4. Read each round aloud against the clock.
  5. Award the streak and lightning bonuses.
🔧 If it is not right, reply with this: If a question can't be done in 20 seconds, reply: "Replace any question that takes longer than 20 seconds with a quicker one, re-verify all answers, and keep the trap questions clearly marked."
THE PROMPT
ROLE: Act as a quiz-show host and mathematics teacher who builds a rapid-fire speed-round game to sharpen recall and mental maths.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. I want a high-energy speed-round game on [TOPIC] where students answer quick questions against the clock, individually or in pairs.

YOUR TASK:
1. Create 4 speed rounds of 10 questions each (40 total) on [TOPIC], every question answerable in under 20 seconds.
2. Make Round 1 easiest and Round 4 hardest; mix in a few 'trap' questions where the obvious answer is wrong.
3. Give exact timing, points per correct answer, and a streak bonus.
4. Provide a clean answer key the teacher can scan while reading aloud.
5. Add 3 'lightning' bonus questions worth double.

INPUT: Topic, class and board are above. If I add nothing, assume a fast oral round with the teacher reading questions.

HOW TO WORK IT OUT: Solve all 40 questions plus the bonuses yourself. Verify every answer by two independent methods and confirm agreement; flag the trap questions clearly. Never invent formulas or values. Keep all maths in clean LaTeX or clear plain text so it can be read aloud without confusion.

OUTPUT FORMAT (use these exact headings):
HOW THE SPEED GAME WORKS (timing, points, streak bonus - numbered)
ROUND 1 (10 questions)
ROUND 2 (10 questions)
ROUND 3 (10 questions)
ROUND 4 (10 questions)
LIGHTNING BONUS (3 questions)
ANSWER KEY (every answer in a box [Answer: ...]; traps marked TRAP)
HOST TIPS (3 bullets)

RULES: Every question must be answerable in under 20 seconds and be fully verified. Difficulty must rise across rounds. No two questions may be near-duplicates. Keep wording crisp for reading aloud.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.
