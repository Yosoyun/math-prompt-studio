Full-Length Timed Mock with Time-Per-Section Plan

A full timed mock plus a minute-by-minute strategy for finishing on time.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Enter class, board, total marks, total time.
  2. 2. List the chapters to include.
  3. 3. Send and review the timing plan.
  4. 4. Ask to adjust if a section is too long.
🔧 If it is not right, reply with this: If timing feels unrealistic, reply: "Re-estimate time per section assuming an average student, and make sure all section times add up to the total time."
THE PROMPT
ROLE: Act as an exam coach and paper-setter who trains students to finish papers on time.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM e.g. CBSE / JEE / NEET / State Board]. Students often run out of time. I want a full-length timed mock AND a clear plan for how long to spend on each part.

YOUR TASK:
1. Build one full-length mock for these chapters: [LIST CHAPTERS]. Total marks: [TOTAL MARKS]. Total time: [TOTAL TIME e.g. 3 hours].
2. Organise it into clear sections by marks.
3. Create a TIMING PLAN: how many minutes to spend per section and a target 'checkpoint clock' (for example, 'by 45 minutes you should be at Section B').
4. Add 3 short time-management tips specific to this paper.
5. Provide a final answer key.

INPUT: Here are the chapters, marks and time: [PASTE DETAILS HERE].

HOW TO WORK IT OUT: Solve all questions internally first and time-budget them realistically for an average [CLASS/GRADE] student. VERIFY every final answer two independent ways. Never invent formulas or values. Make sure the section times add up exactly to the total time. Keep all maths legible in clean plain text or LaTeX.

OUTPUT FORMAT (exact order):
PAPER HEADER: Subject, Class, Board/Exam, Total Marks, Total Time, Instructions.
FULL PAPER: sections A/B/C/D with marks per question.
TIMING PLAN TABLE: Section | No. of questions | Marks | Minutes to spend | Checkpoint clock.
THREE TIMING TIPS for this exact paper.
ANSWER KEY: question number and final answer.
VERIFICATION NOTE: confirm answers and that section times sum to the total.

RULES: Marks must total [TOTAL MARKS]; minutes must total [TOTAL TIME]. Original questions only. Difficulty true to [BOARD or EXAM]. Fix mismatches before showing me.

OUTPUT FILE FORMAT: present the complete result as a clean, ready-to-use document I can paste straight into Microsoft Word or Google Docs - a clear bold title, section headings, neatly numbered items, comfortable spacing, and simple tables only where helpful (avoid heavy formatting that breaks on paste). It must also print cleanly to PDF on A4 (File > Print > Save as PDF). Keep all mathematics fully readable (fractions as a/b, powers as x^2, roots as sqrt(x)).

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block.
