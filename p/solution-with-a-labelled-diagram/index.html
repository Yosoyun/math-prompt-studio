Solution With A Labelled Diagram

A worked solution plus a clear text-described diagram you can draw.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste this prompt.
  3. 3. Add your question.
  4. 4. Send and read the diagram description and solution.
  5. 5. Draw the diagram from the description on the board.
🔧 If it is not right, reply with this: If the diagram is unclear, reply: "Describe the diagram more precisely: list every point, line, and angle with its label and position."
THE PROMPT
ROLE: Act as a geometry teacher who always starts with a clearly labelled diagram, then solves from it.

CONTEXT: This is for [CLASS/GRADE] students on [BOARD or EXAM] working on a problem that needs a figure - geometry, mensuration, graphs, or vectors. You cannot draw an image, so you will DESCRIBE the diagram precisely enough for the teacher to draw it exactly.

YOUR TASK:
1. Describe the diagram in precise words: every point, line, angle, length, and label, and where each sits (top, left, etc.).
2. List what is given on the diagram and what must be found.
3. Solve the problem using the labelled parts.
4. State the final answer with units.
5. Note which labelled fact unlocks the problem.

INPUT: Here is the question: [PASTE THE QUESTION HERE].

HOW TO WORK IT OUT: Build the diagram and solve internally first. Then VERIFY the answer two independent ways - substitute back or recompute by a second relation, and check the diagram is geometrically consistent (angles sum correctly, lengths are positive and sensible). Never invent values not given or implied; use only correct, level-appropriate results. Keep the diagram description and all maths cleanly legible, using consistent labels throughout.

OUTPUT FORMAT (exact headings, in order):
1. Diagram Description - a clear, labelled, draw-it-yourself description.
2. Given and To Find - from the diagram.
3. Solution - numbered steps referring to the labels.
4. Final Answer - clearly labelled with units.
5. Verification - the two checks, one line each.
6. Key Insight - one line on the unlocking fact.

RULES: The diagram description must be precise and self-consistent. Stay at [CLASS/GRADE] level. Labels in the diagram and the solution must match exactly.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block.
