Vector and Force Diagram, Correctly Drawn

A labelled vector or free-body diagram with arrows to scale, directions and the resultant shown.

Makes images - needs an image AI  ·  Best tool:  ChatGPT or Gemini (image)

THE PROMPT
ROLE: Act as a physics-and-maths teacher who draws clean, correctly directed vector and force diagrams.

CONTEXT: I teach [CLASS/GRADE] under [BOARD or EXAM e.g. CBSE / ICSE / JEE / NEET]. Arrow direction, relative length and labelling must all be correct.

YOUR TASK:
1. Draw each vector or force as an arrow starting from the correct point, pointing in the stated direction.
2. Make arrow lengths roughly proportional to their magnitudes so larger forces look larger.
3. Label each arrow with its name and magnitude with units, like F1 = 10 N.
4. If asked, show the resultant as a distinct arrow and label its magnitude and angle.
5. For a free-body diagram, draw the object as a dot or box and show all forces acting on it (weight, normal, tension, friction) from the correct points.

INPUT: Here is the situation: [DESCRIBE IT HERE, e.g. "A block on a horizontal table: weight 50 N down, normal reaction up, applied force 20 N right, friction 5 N left" or "Two vectors from origin: 3 N east and 4 N north, find the resultant"].

HOW TO WORK IT OUT: First list each vector with its magnitude and direction in a small table. If a resultant is needed, compute it on the side (components, then magnitude by Pythagoras, then angle by inverse tangent) and SHOW the working. VERIFY two ways: check that the components of the resultant equal the sums of the individual components, and state whether the system is balanced or has a net force. Never reverse an arrow direction; never invent a force that was not described.

OUTPUT FORMAT:
1. VECTOR TABLE: each vector with magnitude and direction.
2. RESULTANT WORKING: components, magnitude, and angle (if a resultant is asked).
3. THE DIAGRAM: one clean labelled image with arrows to scale and the resultant shown if needed.
4. CHECK: one line confirming the component sums match.

RULES: Arrow directions must match the description exactly. Lengths roughly proportional to magnitudes. Every arrow labelled with name and unit. Plain white background.

SIGNATURE: Sign the bottom-right of every page with an elegant handwritten signature "Indrajeet Yadav" with a small underline flourish.
