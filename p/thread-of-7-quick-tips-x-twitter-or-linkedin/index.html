Thread of 7 Quick Tips (X / Twitter or LinkedIn)

A ready-to-post numbered thread with a hook and closing tweet.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt below.
  3. 3. Add your thread topic.
  4. 4. Send and read the numbered thread.
  5. 5. Post each part in order.
🔧 If it is not right, reply with this: If any tweet is over the limit, reply: "Tweet 4 is too long. Cut it under 280 characters without losing the point."
THE PROMPT
ROLE: Act as a thread writer who turns teaching wisdom into a tight, scroll-stopping social thread.

CONTEXT: I want to post a thread of quick maths-study tips on X (Twitter) or LinkedIn for [CLASS/GRADE] students preparing for [BOARD or EXAM e.g. JEE / NEET / CBSE]. Each part must stand alone yet flow as a series.

YOUR TASK:
1. Write a thread with a hook tweet, 7 tip tweets, and a closing tweet.
2. The hook must promise a clear payoff (for example, '7 tips that fixed my students' [TOPIC] marks').
3. Each numbered tweet delivers ONE specific, usable tip.
4. Keep each tweet under 280 characters.
5. End with a call to action (follow, repost the first tweet, or comment).

INPUT: My thread topic is: [PASTE TOPIC, e.g. how to revise calculus faster / avoiding silly mistakes in exams].

HOW TO WORK IT OUT: If any tip contains a formula, shortcut, or number, VERIFY it is correct before writing it. Never invent rules. Keep maths in clean plain text (^ for powers, sqrt() for roots, / for fractions). Count characters and keep each tweet within 280; if a tip is too long, split or shorten it.

OUTPUT FORMAT: Use this exact structure.
TWEET 1 (Hook): ...
TWEET 2 - Tip 1: ...
TWEET 3 - Tip 2: ...
TWEET 4 - Tip 3: ...
TWEET 5 - Tip 4: ...
TWEET 6 - Tip 5: ...
TWEET 7 - Tip 6: ...
TWEET 8 - Tip 7: ...
TWEET 9 (Closing + CTA): ...
HASHTAGS: 3 to 5 tags to add to the last tweet.
CHARACTER CHECK: confirm every tweet is under 280 characters.

RULES: Every tip correct and useful. No tweet over 280 characters. Punchy, helpful tone. Plain ASCII, straight quotes only.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the header block.
