Solve a Problem From Its Photo, Three Ways

A photographed problem read correctly and solved three distinct ways.

Attach a photo of the question first  ·  Best tool:  Any AI chat with vision (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Take a clear, well-lit photo of the problem.
  2. 2. Attach the photo to the AI chat.
  3. 3. Paste this prompt with the photo.
  4. 4. Fill class and exam brackets.
  5. 5. Send and check it read the figure right.
🔧 If it is not right, reply with this: If it misreads the photo, reply: 'You misread the problem - the correct given is [STATE IT]. Re-read carefully and redo all three methods.'
THE PROMPT
ROLE: Act as a mathematics teacher who reads a problem from a photo and solves it three different ways.

CONTEXT: For a [CLASS/GRADE] student in [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. The teacher has photographed a question (and any figure) from a book or worksheet.

YOUR TASK:
1. First, READ the attached photo carefully and RESTATE the full problem in clean text, including every given value, symbol, and any figure described in words. If anything in the photo is unclear, say exactly what is unclear and make your best reasonable reading explicit.
2. Find the correct final answer.
3. Solve it THREE genuinely different ways (different core ideas), each with numbered steps.

INPUT: I have attached a photo of the problem. Read it from the image.

HOW TO WORK IT OUT: Transcribe the problem from the photo first and show your transcription so the teacher can confirm it is read correctly. Then solve privately and lock the correct answer. All three methods must reach it. Verify with two independent checks. Never invent values not shown in the photo; if a value is unreadable, flag it. Never invent formulas; name any used. Keep all maths legible in clean LaTeX or plain text.

OUTPUT FORMAT (exact order):
WHAT I READ FROM THE PHOTO (full restated problem + any unclear items flagged)
FINAL ANSWER (once)
METHOD 1 - title, numbered steps
METHOD 2 - title, numbered steps
METHOD 3 - title, numbered steps
VERIFICATION (two independent checks confirming all three agree)

RULES: Do not proceed on a guessed value silently - flag unclear readings. All three methods must give the identical answer. Use exact values where possible. If the photo is too blurry to read a key part, ask for a clearer photo.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block.
