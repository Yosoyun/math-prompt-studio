Prerequisite Mind Map (What to Learn First)

A learning-order map showing what a student must know before this topic.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Type the topic students are about to learn.
  3. 3. Paste the prompt below.
  4. 4. Send and read the order bottom-up.
  5. 5. Ask for a quick check question per prerequisite.
🔧 If it is not right, reply with this: If a prerequisite is missing, reply: "Add the foundational skills needed for [sub-topic] and place them in order." If the order is off, reply: "Re-sequence so nothing depends on something taught later."
THE PROMPT
ROLE: Act as a mathematics teacher who maps prerequisites so students learn topics in the right order.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. Before starting a new topic I want a PREREQUISITE MIND MAP (text form) that shows exactly what skills and concepts a student must already have, arranged from the most basic up to the target topic, so I can fill gaps first. No image needed.

YOUR TASK:
1. Put the target topic at the TOP as the goal.
2. Below it, list the direct prerequisites (what is needed immediately before it).
3. Below those, list THEIR prerequisites - go down 2 to 3 levels to the foundations.
4. For each prerequisite add a one-line "why it is needed".
5. End with a READINESS CHECK: 5 quick diagnostic questions to test if a student is ready for the target topic.

INPUT: Here is the target topic: [PASTE TARGET TOPIC HERE]. If only a name is given, use the standard [BOARD] curriculum sequence for [CLASS/GRADE].

HOW TO WORK IT OUT: First trace the true dependency chain privately - make sure nothing listed actually depends on the target topic itself (no circular order). VERIFY each prerequisite link twice for correctness and ordering. Never invent a dependency; only list genuine ones. Keep all maths in clean plain text.

OUTPUT FORMAT (exactly this, read top to bottom):
GOAL TOPIC: <name>
   needs ->
LEVEL 1 PREREQUISITES (just before the goal):
   - <skill> : why needed
   - <skill> : why needed
      needs ->
LEVEL 2 PREREQUISITES:
   - <skill> : why needed
      needs ->
LEVEL 3 / FOUNDATIONS:
   - <skill> : why needed
READINESS CHECK (5 quick questions): 1... 2... 3... 4... 5...

RULES: No circular dependencies. Order must be genuinely bottom-up. Match to [CLASS/GRADE].

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.

SIGNATURE: End your ENTIRE response with exactly this footer block (keep the lines and symbols):
