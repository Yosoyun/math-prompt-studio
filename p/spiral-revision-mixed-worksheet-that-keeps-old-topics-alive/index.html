Spiral Revision Mixed Worksheet That Keeps Old Topics Alive

A mixed worksheet that revisits several past chapters in a deliberate spiral, with a topic map and answer key.

Works on any free AI  ·  Best tool:  ChatGPT (free) or Claude (free)

✅ How to use this effectively

  1. 1. List every chapter taught so far.
  2. 2. Set the weighting (recent vs older).
  3. 3. Paste the class and exam board.
  4. 4. Copy to Word and print.
  5. 5. Use the topic map to spot weak areas.
🔧 If it is not right, reply with this: If it leans too heavily on one chapter, type: 'Rebalance so no chapter has more than 30 percent of questions; interleave them, do not block them.'
THE PROMPT
Act as a master mathematics teacher who builds spiral-revision worksheets that fight forgetting through interleaving.

CONTEXT: Class is [CLASS/GRADE] under [BOARD or EXAM]. The chapters covered so far this term are: [LIST CHAPTERS, e.g. fractions, linear equations, ratio, mensuration]. I want a single mixed worksheet that deliberately interleaves these topics rather than grouping them, so students must first decide which method applies.

YOUR TASK:
1. Build one worksheet of 15 questions that mixes the listed chapters in non-blocked order (no two consecutive questions from the same chapter where possible).
2. Weight roughly [WEIGHTING, e.g. 60 percent recent chapters, 40 percent older] so older topics stay alive but recent learning dominates.
3. Include a spread of difficulty: about 5 warm-up, 7 standard, 3 challenge.
4. Tag each question silently with its chapter, then reveal the tags only in the teacher map at the end.
5. End with a short 'Topic Map' table: chapter vs question numbers, so I can see coverage at a glance.

INPUT: Chapters, class and weighting are above. If I paste last week's quiz here: [PASTE or leave blank], avoid repeating those exact questions.

HOW TO WORK IT OUT: Solve each question internally before finalising it. VERIFY every answer with two independent checks (re-solve by an alternate route or substitute back). Never invent formulas or fudge numbers; keep answers clean. Keep all mathematics legible in plain text (^ for powers, sqrt(), / for fractions, clear units).

OUTPUT FORMAT (exact order):
- Title block with 'Compiled by Indrajeet Yadav', term/week label, and Name / Class / Date lines.
- Instructions: total marks, suggested time, 'identify the topic before solving'.
- Questions 1-15 (interleaved, mixed difficulty).
- '--- ANSWER KEY ---' with final answers and a one-line method hint per question.
- 'TOPIC MAP' table: chapter | question numbers | count.

RULES: Genuine interleaving, not blocked sections. Honest weighting. A4-friendly, spaced for working. No invented content; stay within the listed chapters. State one assumption if anything is unclear and proceed.

OUTPUT FILE FORMAT: present the complete result as a clean, ready-to-use document I can paste straight into Microsoft Word or Google Docs - a clear bold title, section headings, neatly numbered items, comfortable spacing, and simple tables only where helpful (avoid heavy formatting that breaks on paste). It must also print cleanly to PDF on A4 (File > Print > Save as PDF). Keep all mathematics fully readable (fractions as a/b, powers as x^2, roots as sqrt(x)).

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: 'Prepared by Indrajeet Yadav', and put 'Compiled by Indrajeet Yadav' in the title block.
