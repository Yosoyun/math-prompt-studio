Single-Chapter Unit Test (30 to 40 marks) with Key

A focused unit test on one chapter, balanced by difficulty, with a clean answer key.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Paste the prompt into a fresh chat.
  2. 2. Type your single chapter name.
  3. 3. Set the marks and time.
  4. 4. Send and read the test.
  5. 5. Check the key at the bottom.
🔧 If it is not right, reply with this: If it strays beyond the chapter, reply: 'Question [number] is outside the chapter [chapter name]. Replace it with a question only from this chapter and re-verify the answer.'
THE PROMPT
ROLE: Act as an experienced mathematics teacher who sets crisp, fair unit tests.

CONTEXT: This is a class unit test for Class [CLASS/GRADE], [BOARD or EXAM e.g. CBSE / ICSE / State Board], on ONE chapter only: [CHAPTER NAME]. Total marks [TOTAL MARKS e.g. 30], time [TIME e.g. 1 hour 15 minutes]. It will be used in the next lesson, so it must be self-contained.

YOUR TASK:
1. Cover only the sub-topics of this one chapter, spread evenly so no sub-topic is missed.
2. Use a difficulty mix of [e.g. 50 percent easy, 30 percent moderate, 20 percent hard].
3. Include a small mix of question types: short-answer, computation and one application question.
4. Provide a complete answer key.

INPUT: The chapter is: [CHAPTER NAME]. The key sub-topics to test are: [LIST SUB-TOPICS, or write 'use the standard sub-topics'].

HOW TO WORK IT OUT: Solve each question internally first, then verify every final answer twice using two independent methods. Never invent formulas or values. Keep all mathematics legible in clean LaTeX or clear plain text. Make sure marks total exactly [TOTAL MARKS] before printing.

OUTPUT FORMAT, in this order:
1. HEADER: 'Unit Test - [CHAPTER NAME]', 'Class: [CLASS/GRADE]', 'Time: [TIME]', 'Maximum Marks: [TOTAL MARKS]', 'Compiled by Indrajeet Yadav'.
2. GENERAL INSTRUCTIONS: 3 to 5 numbered lines.
3. THE TEST: numbered questions with marks shown beside each.
4. ANSWER KEY: final answer for every question, with a short 1 to 2 line working note for harder ones.

RULES: Stay strictly inside [CHAPTER NAME]. No ambiguous wording. Number every question. Marks must total exactly.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: 'Prepared by Indrajeet Yadav', and put 'Compiled by Indrajeet Yadav' in the header.
