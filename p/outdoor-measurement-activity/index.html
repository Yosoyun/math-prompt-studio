Outdoor Measurement Activity

A safe outdoor task to measure real heights/distances, with the maths and a recording sheet.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Name what to measure and the class.
  2. 2. Paste the prompt into any AI chat.
  3. 3. Read the safety notes and steps.
  4. 4. Print the recording sheet.
  5. 5. Run it on the school ground.
🔧 If it is not right, reply with this: If the method needs tools you lack, reply: "My students only have a measuring tape and a protractor. Redesign the method using only these and re-verify the example."
THE PROMPT
ROLE: Act as a maths teacher who runs safe, practical outdoor measurement activities on the school grounds.

CONTEXT: I teach [CLASS/GRADE] under [BOARD e.g. CBSE / ICSE / State Board]. I want an OUTDOOR MEASUREMENT ACTIVITY where students measure a real [TARGET e.g. height of a tree or building / width of a field / distance across a gap] using [MATHS METHOD e.g. similar triangles / trigonometry / a homemade clinometer / shadow ratios] and only simple tools.

YOUR TASK:
1. State what students will measure and the maths method used.
2. List the simple tools needed.
3. Give numbered field steps, including how to take each reading.
4. Provide a recording sheet (table) for measurements.
5. Show the full calculation with a worked example using sample readings.
6. List clear safety rules for being outdoors.
7. Provide a 15-mark rubric (accuracy, method, teamwork).

INPUT: Here is my target and the tools we have: [PASTE TARGET AND TOOLS HERE]. If blank, assume a measuring tape and a protractor and choose a suitable method for the class and board above.

HOW TO WORK IT OUT: Plan internally and ensure the method genuinely gives the measurement with the listed tools. VERIFY the worked example two independent ways (for example, by the chosen method and by a rough estimate or alternate ratio) and show both. Never invent formulas. Keep all maths legible in clean plain text or simple LaTeX.

OUTPUT FORMAT (use these exact headings in this order):
1. ACTIVITY TITLE
2. WHAT WE MEASURE AND HOW (method named)
3. TOOLS
4. SAFETY RULES
5. FIELD STEPS (numbered)
6. RECORDING SHEET (table headings, empty rows)
7. WORKED EXAMPLE WITH TWO CHECKS
8. RUBRIC (15 marks)

RULES: Safety first; keep students away from roads and hazards. Be exact with units and readings. Short, plain sentences.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title block at the top.
