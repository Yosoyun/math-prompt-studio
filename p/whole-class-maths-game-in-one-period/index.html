Whole-Class Maths Game in One Period

A complete, named classroom game with setup, 3 rising rounds of verified questions, scoring and a tie-breaker.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. Type your topic, class and board.
  2. Add the period length and class size.
  3. Paste the prompt and send.
  4. Read setup aloud, then play the rounds.
  5. Use the tie-breaker if scores are level.
🔧 If it is not right, reply with this: If any question is too hard or the timing feels off, reply: "Re-check question [number], verify its answer a second way, and make Round 1 easier so it fits the time."
THE PROMPT
ROLE: Act as a master mathematics teacher and game designer who turns dry topics into a fast, fair, whole-class game.

CONTEXT: I teach [CLASS/GRADE] students preparing for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. I want a ready-to-run classroom maths game on the topic [TOPIC] for a [NUMBER]-minute period with about [NUMBER] students seated in teams.

YOUR TASK:
1. Design ONE complete, original classroom game with a catchy name that drills [TOPIC].
2. Write clear setup instructions a teacher can follow in 2 minutes with only a board, chalk and paper.
3. Create 15 game questions sorted into 3 rounds: Warm-up (easy), Core (medium), Challenge (hard).
4. Give exact scoring, timing per question, and one tie-breaker question.
5. Add 3 simple rules to keep it fair and noise under control.

INPUT: Here is what to use: topic, class, board, period length and class size are given above. If I add nothing else, assume a normal classroom with no projector.

HOW TO WORK IT OUT: Solve every question yourself first. Verify each final answer by two independent methods (for example re-substitution and a quick estimate) and confirm they agree. Never invent formulas or fake values. Keep all maths in clean LaTeX or clear plain text so it reads correctly on a board.

OUTPUT FORMAT (use these exact headings in this order):
GAME NAME AND ONE-LINE PITCH
WHAT YOU NEED (materials)
HOW TO SET UP (numbered, under 2 minutes)
HOW TO PLAY (numbered rules)
ROUND 1 - WARM-UP (5 questions with answers)
ROUND 2 - CORE (5 questions with answers)
ROUND 3 - CHALLENGE (5 questions with answers)
SCORING AND TIMING TABLE
TIE-BREAKER QUESTION (with answer)
TEACHER TIPS (3 bullets)

RULES: Every question must be solvable in the stated time. Show the final answer for each in a box like [Answer: ...]. No two questions may be near-duplicates. Difficulty must clearly rise across rounds. Keep language simple enough for a teenager to read aloud.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block.
