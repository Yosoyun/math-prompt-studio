Three Parallel Model Paper Set (Same Level)

Three equal-difficulty model papers (Set A, B, C) with one shared answer key.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Enter class, board, and chapter list.
  2. 2. State total marks for each set.
  3. 3. Send and wait for all three sets.
  4. 4. Ask for the combined key if missing.
  5. For a trustworthy answer key: paste the finished paper into a FRESH chat and ask it to solve and grade it as an unseen exam (it catches its own slips).
🔧 If it is not right, reply with this: If one set feels harder, reply: "Set B is harder than A and C - rebalance so all three sets are equal difficulty, then show only the changed questions."
THE PROMPT
ROLE: Act as an examination board paper-setter who designs matched parallel papers.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE]. I want THREE different model papers (Set A, Set B, Set C) that are equal in difficulty, so I can give different rows different papers and stop copying.

YOUR TASK:
1. Create three full model papers covering the same chapters: [LIST CHAPTERS]. Each paper is worth [TOTAL MARKS] and follows the same section pattern.
2. The three sets must be PARALLEL: the question in position 1 of each set must test the same idea at the same difficulty, just with different numbers or wording. Do the same for every position.
3. Make all questions original. Do not reuse a question across the three sets.
4. Produce ONE combined answer key listing final answers for Set A, B and C side by side.

INPUT: Here are the chapters and marks: [PASTE CHAPTERS AND MARKS HERE].

HOW TO WORK IT OUT: Solve every question in all three sets internally first. VERIFY each final answer by two independent methods (back-substitution, second method, or estimate-and-compare). Never invent formulas or values. Confirm the three sets really are equal in difficulty before showing them. Keep all maths fully legible in clean plain text or LaTeX.

OUTPUT FORMAT (exact order):
SET A - full paper with sections and marks.
SET B - full paper, same structure.
SET C - full paper, same structure.
DIFFICULTY-MATCH TABLE: for each question position, one row showing 'tests the same skill across A/B/C'.
COMBINED ANSWER KEY: a table - Question No. | Set A answer | Set B answer | Set C answer.
VERIFICATION NOTE: confirm answers were double-checked.

RULES: All three sets must total exactly [TOTAL MARKS]. No question repeats across sets. Difficulty must match a real [BOARD or EXAM] paper. Fix any imbalance before showing me.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block.
