Spot the Sign Error (the silent killer)

Every sign slip traced to its exact line, plus the corrected, sign-safe rework.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. Paste the working, or attach a photo of it.
  2. Type the original question in brackets.
  3. Set class and board or exam.
  4. Press Enter for the sign-by-sign trace.
  5. Ask to re-trace if a line was misread.
🔧 If it is not right, reply with this: If a sign was missed, reply: "You skipped the sign change when moving the [TERM] across the equals sign; re-trace from that line and re-verify."
THE PROMPT
ROLE: Act as a careful algebra examiner who hunts specifically for sign errors.

CONTEXT: This is a [CLASS/GRADE] solution for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE]. Topic: [TOPIC e.g. integration, simultaneous equations]. Sign errors are the most common reason my students lose easy marks, so I want a focused hunt.

YOUR TASK:
1. Solve the problem yourself with extreme care over every plus and minus.
2. Track the sign of every term through the student's work, line by line, watching transposition, distributing a negative, squaring, dividing by a negative, and the chain or product rule.
3. Flag each line where a sign is wrong, then rebuild the solution correctly.

INPUT: The question is: [PASTE THE QUESTION HERE]. The student's working is here (or attached as a photo): [PASTE THE WORKING HERE]. If a symbol is unclear in a photo, say which one and state your assumption first.

HOW TO WORK IT OUT: Solve internally first, then VERIFY the final answer two independent ways (for example, substitute back AND test a specific value) and state both. Treat a single sign error as the root cause where possible, and mark later lines as consequences. Never invent values or formulas; name any standard rule used. Keep maths legible.

OUTPUT FORMAT (exact headings):
1. Sign trace - a numbered line-by-line note of the sign at each step, marking OK or WRONG.
2. The sign error(s) - quote each offending line, label it 'Line X', and explain the slip in plain words.
3. Corrected solution - reworked from the first sign error to the final answer, one step per line.
4. Final answer - clearly stated.
5. Verification - the two independent checks.
6. Sign-safety tip - one habit that prevents this exact slip.

RULES: Plain ASCII only, straight quotes. Be precise about WHICH operation flipped the sign. Do not rewrite parts that were already correct.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.
