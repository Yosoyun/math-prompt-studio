Worked-Example-Then-Practice Sheet for Confident Self-Study

Paired example-and-practice blocks: each new idea gets a model solution then matched questions, with a key.

Works on any free AI  ·  Best tool:  ChatGPT (free) or Claude (free)

✅ How to use this effectively

  1. 1. Set topic, class and board.
  2. 2. List the sub-skills in order.
  3. 3. Check each model before printing.
  4. 4. Give it for self-study or absentees.
  5. 5. Use the key for self-marking.
🔧 If it is not right, reply with this: If the practice questions don't match the example, type: 'Make each practice block use the exact same method as its worked example, just with new numbers; no surprises.'
THE PROMPT
Act as a mathematics teacher and textbook author designing a self-teaching 'worked example then matched practice' worksheet.

CONTEXT: Class [CLASS/GRADE], board/exam [BOARD or EXAM], topic [TOPIC, e.g. simultaneous equations]. I want a sheet a student can learn from alone (useful for absentees, substitute lessons, or home study), where each new sub-skill is taught by a clean worked example and then drilled with matched practice.

YOUR TASK:
1. Break [TOPIC] into [NUMBER, e.g. 4] sub-skills in a sensible teaching order (list them as [SUB-SKILLS] or let me leave it blank for you to choose).
2. For each sub-skill create a block: one fully worked EXAMPLE (numbered steps, short reasons) followed by 4 PRACTICE questions that use the exact same method with fresh numbers.
3. Difficulty within each block should rise slightly from question 1 to 4.
4. End with a MIXED SET of 5 questions drawing on all sub-skills together.
5. Make the worked examples genuinely model the practice; no hidden new steps in practice that the example did not show.

INPUT: Topic, class, board (and optional sub-skill list) are above. If I paste my house style for solutions here: [PASTE or leave blank], match its layout.

HOW TO WORK IT OUT: Solve everything internally first. VERIFY each example and each answer by two independent checks (substitute back and/or alternate method). Never invent rules or values; keep numbers and answers clean. Keep mathematics fully legible in plain text (^ for powers, sqrt(), / for fractions, aligned steps).

OUTPUT FORMAT (exact order):
- Title block with 'Compiled by Indrajeet Yadav', topic, Name / Class / Date lines.
- 'How to use this sheet' (2 lines: read the example, then do the matched practice).
- For each sub-skill: 'EXAMPLE n' (worked) then 'PRACTICE n' (4 questions).
- 'MIXED SET' (5 questions).
- '--- ANSWER KEY ---' with final answers for all practice and mixed questions.

RULES: Examples must fully model their practice. Clean numbers. Self-explanatory enough for solo study. A4-friendly with working space. No invented mathematics. State one assumption if unclear and continue.

OUTPUT FILE FORMAT: present the complete result as a clean, ready-to-use document I can paste straight into Microsoft Word or Google Docs - a clear bold title, section headings, neatly numbered items, comfortable spacing, and simple tables only where helpful (avoid heavy formatting that breaks on paste). It must also print cleanly to PDF on A4 (File > Print > Save as PDF). Keep all mathematics fully readable (fractions as a/b, powers as x^2, roots as sqrt(x)).

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: 'Prepared by Indrajeet Yadav', and put 'Compiled by Indrajeet Yadav' in the title block.
