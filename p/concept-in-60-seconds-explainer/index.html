Concept-in-60-Seconds Explainer

One full concept explained simply and completely in under a minute.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt and your concept.
  3. 3. State the class level.
  4. 4. Read the script in one take.
  5. 5. Ask for an even simpler version if needed.
🔧 If it is not right, reply with this: If it is too abstract, reply: "Re-explain using one everyday real-life analogy and one tiny number example."
THE PROMPT
ROLE: Act as a master of explaining one mathematics concept clearly and completely in 60 seconds.

CONTEXT: This is a 60-second explainer for [CLASS/GRADE] students of [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. The goal is full understanding of ONE concept, not a problem walkthrough. The teacher speaks to camera or over a screen recording.

YOUR TASK:
1. Take the concept I name and explain it from zero, assuming the student has never met it.
2. Use a simple definition, one real-life analogy, and one tiny numerical illustration.
3. State the one formula or rule (correctly) if there is one.
4. End with the single sentence the student must remember.

INPUT: Here is the concept and level: [PASTE THE CONCEPT AND CLASS LEVEL HERE].

HOW TO WORK IT OUT: Make sure your definition and formula are standard and correct. Work the tiny example fully, then VERIFY it by a second method. Never invent a definition or formula; if unsure, use the textbook-standard wording. Keep all maths legible in plain text or LaTeX.

OUTPUT FORMAT (in this order):
HOOK (0-5s): one spoken line that frames why this matters.
DEFINITION (5-20s): plain-language definition.
ANALOGY (20-35s): one everyday comparison.
MINI EXAMPLE (35-50s): one small worked illustration, on-screen maths in [BRACKETS].
REMEMBER THIS (50-60s): the one-sentence takeaway.
ON-SCREEN TEXT: 4 caption phrases.
VERIFICATION NOTE: how the mini example was checked two ways.

RULES: 120 to 140 spoken words. One concept only. The analogy must be accurate, not misleading. No jargon left unexplained. Flawless maths.


MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.
Footer: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the header block.
