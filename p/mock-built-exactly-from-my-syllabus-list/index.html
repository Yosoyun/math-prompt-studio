Mock Built Exactly From My Syllabus List

A mock that stays strictly inside the exact syllabus you paste, nothing extra.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Paste your exact syllabus or topic list.
  2. 2. State total marks and time.
  3. 3. Send the prompt.
  4. 4. Check the coverage table against your list.
🔧 If it is not right, reply with this: If it adds out-of-syllabus content, reply: "Question 3 uses a topic not in my syllabus list - replace it with one that stays strictly inside what I pasted."
THE PROMPT
ROLE: Act as a syllabus-faithful paper-setter who never strays outside the prescribed topics.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM e.g. CBSE / State Board / a coaching syllabus]. My syllabus has specific included topics and some EXCLUDED ones. I need a mock that stays strictly inside what I give you - nothing extra, nothing deleted.

YOUR TASK:
1. Read my syllabus list carefully. Build a full mock that covers ONLY the listed topics, ideally touching each at least once. Total marks: [TOTAL MARKS]. Time: [TIME].
2. Do NOT use any topic, formula or method that is not in my list, even if it would normally appear in this exam.
3. Produce a COVERAGE TABLE mapping each syllabus item to the question(s) that test it, so I can confirm nothing is missed or added.
4. Provide solutions and an answer key.

INPUT: Here is my exact syllabus / topic list: [PASTE YOUR SYLLABUS LIST HERE]. Anything NOT on this list is out of bounds.

HOW TO WORK IT OUT: Solve everything internally first. VERIFY each answer two independent ways. Never invent formulas or values, and never use a method outside my list. Before finishing, re-read each question and confirm it needs ONLY listed topics. Keep all maths legible in clean plain text or LaTeX.

OUTPUT FORMAT (exact order):
HEADER: Subject, Class, Board/Exam, Marks, Time.
PAPER: sections, numbered questions.
COVERAGE TABLE: Syllabus item | Question number(s) testing it | Covered? (Yes).
OUT-OF-SYLLABUS CHECK: one line confirming no question uses anything off my list.
SOLUTIONS: full steps.
ANSWER KEY: compact list.
VERIFICATION NOTE: confirm answers checked and syllabus respected.

RULES: Strictly inside my pasted list. Cover each item at least once if possible. Marks total [TOTAL MARKS]. Original questions.

OUTPUT FILE FORMAT: present the complete result as a clean, ready-to-use document I can paste straight into Microsoft Word or Google Docs - a clear bold title, section headings, neatly numbered items, comfortable spacing, and simple tables only where helpful (avoid heavy formatting that breaks on paste). It must also print cleanly to PDF on A4 (File > Print > Save as PDF). Keep all mathematics fully readable (fractions as a/b, powers as x^2, roots as sqrt(x)).

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block.
