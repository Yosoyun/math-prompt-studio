Table of Contents / Index for a Whole Book

A clean, ordered table of contents with chapters and sub-sections.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. List your chapters or the subject and class.
  2. 2. State the board or exam.
  3. 3. Send the prompt.
  4. 4. Reorder chapters to match your plan.
🔧 If it is not right, reply with this: If the order is poor, reply: "Reorder the chapters by teaching dependency and tell me which chapter each one builds on."
THE PROMPT
ROLE: Act as a mathematics textbook editor preparing the table of contents.

CONTEXT: I am compiling a book for [CLASS/GRADE] students of [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET] on [SUBJECT/SCOPE e.g. full Class 10 Mathematics]. I need a professional, logically ordered table of contents with chapters and sub-sections.

YOUR TASK:
1. Produce a complete table of contents in teaching order.
2. Number the chapters and give each a clear, standard title.
3. Under each chapter, list 3-6 sub-sections.
4. Group chapters into Units or Parts if that helps clarity.
5. Add standard front matter (Preface, How To Use) and back matter (Formula Appendix, Answers, Glossary, Index) entries.
6. Leave a clear placeholder for page numbers.

INPUT: Here is my chapter list or syllabus (or leave blank to use the standard one): [PASTE HERE].

HOW TO WORK IT OUT: Decide the dependency order internally first. VERIFY two ways: (a) check each chapter's prerequisites appear earlier in the list, and (b) check the contents fully cover the stated syllabus with no gaps or extras. Do not invent non-syllabus chapters. Keep titles clean and standard.

OUTPUT FORMAT (exact order):
1. Heading: "Contents"
2. Front Matter entries
3. Units/Parts with numbered chapters and indented sub-sections
4. Back Matter entries (appendices, answers, glossary, index)
5. Note line: "[Page numbers to be filled at layout stage]"

RULES: Logical dependency order is the priority. Consistent numbering and indentation. Standard chapter titles. Nothing outside the stated scope.

OUTPUT FILE FORMAT: present the complete result as a clean, ready-to-use document I can paste straight into Microsoft Word or Google Docs - a clear bold title, section headings, neatly numbered items, comfortable spacing, and simple tables only where helpful (avoid heavy formatting that breaks on paste). It must also print cleanly to PDF on A4 (File > Print > Save as PDF). Keep all mathematics fully readable (fractions as a/b, powers as x^2, roots as sqrt(x)).

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the header block at the top.
