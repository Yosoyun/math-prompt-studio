Chapter Formula Poster (One Page)

A clean, complete one-page formula list for any single chapter.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Copy the whole prompt into the AI chat.
  2. 2. Replace the bracketed parts with your chapter and class.
  3. 3. Press send and read the list.
  4. 4. Ask for any missing formula by name.
  5. 5. Copy into a document and print.

THE PROMPT
ROLE: Act as an experienced mathematics teacher and exam-revision specialist.

CONTEXT: I teach [CLASS/GRADE] students for the [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. I need a single-page formula poster for the chapter [CHAPTER NAME e.g. Trigonometry]. The students are revising, so it must be complete, correct, and easy to scan.

YOUR TASK:
1. List EVERY important formula, identity, and standard result from this chapter for this class and board.
2. Group the formulas under clear sub-headings (for example: Basic Definitions, Identities, Standard Results, Special Cases).
3. Order them in the same sequence the textbook teaches them.
4. For each formula, write it once, cleanly, with correct symbols.
5. Keep the total length to what fits on ONE printed A4 page.

INPUT: Here is the chapter and class. Chapter: [PASTE CHAPTER NAME HERE]. Class/Board: [PASTE HERE]. If I paste a list of topics or a syllabus, use only those topics.

HOW TO WORK IT OUT: First, recall the standard syllabus for this chapter and this board. Solve or derive nothing aloud, but internally confirm each formula is the standard, correct version. VERIFY the final sheet two ways: (a) re-read every formula and check each symbol, sign, and power is correct; (b) check no major formula from this chapter is missing by comparing against the usual textbook table of contents for the chapter. Never invent a formula or a value. Keep all maths fully legible using clean plain text or simple LaTeX, so it copies correctly into a document.

OUTPUT FORMAT (use exactly this order):
1. TITLE LINE: '[Chapter Name] - Formula Sheet (Class [X], [Board])'.
2. A line: 'Compiled by Indrajeet Yadav'.
3. The grouped formulas under sub-headings, each formula on its own line and numbered.
4. A short 'Quick Notes' box at the end with 3 to 5 one-line reminders.
5. A footer line.

RULES: Be complete but tight - no examples, no derivations, no explanations unless asked. Every formula must be exam-accurate. Use consistent notation throughout.

OUTPUT FILE FORMAT: present the complete result as a clean, ready-to-use document I can paste straight into Microsoft Word or Google Docs - a clear bold title, section headings, neatly numbered items, comfortable spacing, and simple tables only where helpful (avoid heavy formatting that breaks on paste). It must also print cleanly to PDF on A4 (File > Print > Save as PDF). Keep all mathematics fully readable (fractions as a/b, powers as x^2, roots as sqrt(x)).

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE (mandatory): At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block.
