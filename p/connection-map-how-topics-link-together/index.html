Connection Map: How Topics Link Together

A clear map of how one topic connects to others across the syllabus.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt.
  3. 3. Enter the topic and exam.
  4. 4. Send and read the connections.
  5. 5. Share the big-picture line with your class.
🔧 If it is not right, reply with this: If a link is weak, reply: "Drop connection 3 and add a stronger link to [topic], explaining exactly how the two ideas feed each other."
THE PROMPT
ROLE: Act as a mathematics teacher who shows students how every topic connects to the others.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. Students see this topic in isolation: [WRITE THE TOPIC HERE]. I want to reveal how it connects to other topics so it stops feeling like a random island.

YOUR TASK:
1. Identify five to seven other topics this one connects to.
2. For each, explain the connection in one or two plain sentences (what flows from one to the other).
3. Label each link as USES (needs the other) or POWERS (enables the other).
4. Describe one exam-style problem that combines this topic with another.
5. Give one big-picture sentence a teacher can say to show the web of ideas.

INPUT: Here is the central topic: [PASTE OR TYPE IT HERE]. Class or exam: [CLASS/GRADE] / [BOARD or EXAM].

HOW TO WORK IT OUT: Internally confirm each connection is real and used in this syllabus, not a vague resemblance. VERIFY the combined problem by solving it fully and checking the answer two independent ways. Never invent fake links or formulas. Keep maths legible in clear plain text.

OUTPUT FORMAT (use these exact headings):
- CENTRAL TOPIC
- CONNECTIONS (a list; each: linked topic | USES or POWERS | the link in 1 to 2 sentences)
- A PROBLEM THAT COMBINES TWO TOPICS (statement, full solution, and VERIFICATION)
- THE BIG PICTURE IN ONE SENTENCE

RULES: Only genuine, syllabus-relevant connections. Each link must say what actually flows between the topics, not just that they are related. Keep it concrete.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block.
