Solution Graded Out Of The Marks

A model answer plus a marking rubric for grading student scripts.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste this prompt.
  3. 3. Add the question and total marks.
  4. 4. Send and read the model answer and rubric.
  5. 5. Use the rubric to grade student scripts fairly.
🔧 If it is not right, reply with this: If partial-credit rules are unclear, reply: "Spell out the partial marks for a correct method with a wrong final answer."
THE PROMPT
ROLE: Act as a head examiner who writes both the model answer and the marking rubric used to grade student scripts.

CONTEXT: This is for a [CLASS/GRADE] teacher on [BOARD or EXAM] who must grade many scripts for this question fairly and consistently.

YOUR TASK:
1. Write the model answer in numbered steps.
2. Build a marking rubric that splits the total marks across the steps.
3. State partial-credit rules clearly - for example, marks for correct method even if the final answer is wrong.
4. List acceptable alternative methods that should also earn full marks.
5. Note common wrong answers and how many marks, if any, they earn.

INPUT: Here is the question and its total marks. Question: [PASTE THE QUESTION HERE]. Total marks: [TOTAL e.g. 5].

HOW TO WORK IT OUT: Solve internally first. Then VERIFY the model answer two independent ways - substitute back, and recompute by a second route or estimate. Make the marks add exactly to the total and the partial-credit rules unambiguous. Never invent formulas or values; use only correct, level-appropriate maths. Keep all maths and the rubric cleanly legible.

OUTPUT FORMAT (exact headings, in order):
1. Model Answer - numbered steps; final answer with units.
2. Marking Rubric - table: Step | What earns the mark | Marks; with the total.
3. Partial-Credit Rules - short bullets, including method-right answer-wrong.
4. Acceptable Alternative Methods - short bullets.
5. Common Wrong Answers and Marks - short bullets.

RULES: Marks must total exactly the given marks. Rules must be unambiguous for fair grading. Stay at [CLASS/GRADE] level.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block.
