Sectioned Paper A/B/C/D with Marks per Section

A four-section paper (MCQ, short, long, case-study) with marks per section and a full key.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Paste the prompt into a chat.
  2. 2. Set marks for sections A, B, C, D.
  3. 3. List the chapters to cover.
  4. 4. Send and check each section header.
  5. 5. Verify the grand total equals your target.
  6. For a trustworthy answer key: paste the finished paper into a FRESH chat and ask it to solve and grade it as an unseen exam (it catches its own slips).
🔧 If it is not right, reply with this: If a section is too easy or too hard, reply: 'Section [letter] difficulty is off. Re-write Section [letter] to be [easier/harder] while keeping its marks the same, then re-verify the key.'
THE PROMPT
ROLE: Act as a mathematics examiner who builds well-structured sectioned papers.

CONTEXT: This paper is for Class [CLASS/GRADE], [BOARD or EXAM e.g. CBSE / ICSE / State Board]. It must have FOUR sections: Section A (MCQ / very short), Section B (short answer), Section C (long answer), Section D (case-study or application). Total marks [TOTAL MARKS], time [TIME].

YOUR TASK:
1. Distribute marks as: Section A [A MARKS], Section B [B MARKS], Section C [C MARKS], Section D [D MARKS]. (If I leave these blank, choose a sensible split and state it.)
2. Increase difficulty gradually from A to D.
3. In Section D, write a short real-world scenario followed by 2 to 4 sub-questions.
4. Provide a complete answer key by section.

INPUT: Chapters or topics to cover: [LIST CHAPTERS or TOPICS]. Use only these.

HOW TO WORK IT OUT: Solve every question internally first, then verify each final answer twice by independent methods. Never invent formulas or values. Keep all mathematics legible in clean LaTeX or plain text. Confirm A + B + C + D equals [TOTAL MARKS] exactly before printing; fix it if not.

OUTPUT FORMAT, in this order:
1. HEADER BLOCK: paper title, 'Class: [CLASS/GRADE]', 'Board: [BOARD or EXAM]', 'Time: [TIME]', 'Maximum Marks: [TOTAL MARKS]', 'Compiled by Indrajeet Yadav'.
2. GENERAL INSTRUCTIONS: numbered, noting the marks each section carries.
3. SECTION A, SECTION B, SECTION C, SECTION D: each with its heading, its total marks, and per-question marks.
4. ANSWER KEY: organised by section, final answers plus short outlines for long answers.

RULES: Stay inside the listed syllabus. Each section heading must show its total marks. Number every question. Grand total exact.

OUTPUT FILE FORMAT: present the complete result as a clean, ready-to-use document I can paste straight into Microsoft Word or Google Docs - a clear bold title, section headings, neatly numbered items, comfortable spacing, and simple tables only where helpful (avoid heavy formatting that breaks on paste). It must also print cleanly to PDF on A4 (File > Print > Save as PDF). Keep all mathematics fully readable (fractions as a/b, powers as x^2, roots as sqrt(x)).

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.
