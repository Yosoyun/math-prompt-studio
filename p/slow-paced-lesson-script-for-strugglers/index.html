Slow-Paced Lesson Script for Strugglers

A minute-by-minute slow lesson plan with pauses, checks and re-teaching built in.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt.
  3. 3. Enter topic, class and minutes available.
  4. 4. Send and read the script.
  5. 5. Teach directly from it.
🔧 If it is not right, reply with this: If it is rushed, reply: "Slow it down more, add extra pauses and a re-teach moment, and re-verify the examples."
THE PROMPT
ROLE: Act as a remedial maths lesson designer who writes slow, patient, minute-by-minute lessons for weak students who need extra time.

CONTEXT: I am teaching a small remedial group in [CLASS/GRADE] under [BOARD or EXAM e.g. CBSE / ICSE / State Board]. They need a much slower pace than normal class, with frequent pauses and checks. I have [NUMBER] minutes for this lesson.

YOUR TASK:
1. Write a slow lesson on the topic that fits exactly the minutes I have.
2. Lay it out minute-by-minute (or in small time blocks) with a clear teacher action for each.
3. Build in regular pauses, simple 'thumbs up / thumbs down' checks, and ONE planned re-teach moment in case they look lost.
4. Use only one tiny example at a time, fully worked.
5. End with a 2-minute easy exit check.

INPUT: Here is the topic and the minutes I have: [PASTE TOPIC AND MINUTES].

HOW TO WORK IT OUT: Solve every example internally and VERIFY each answer two independent ways before including it. Make sure the time blocks add up exactly to the minutes I gave. Never invent formulas or values. Keep all maths clean and legible.

OUTPUT FORMAT (exact order):
1) Header: 'Slow Lesson - [Topic]' with 'Compiled by Indrajeet Yadav'.
2) LESSON GOAL - one simple line.
3) MINUTE-BY-MINUTE PLAN - a table: Time | Teacher Does | Students Do | Check.
4) WORKED EXAMPLES - each tiny example, step by step.
5) IF THEY LOOK LOST - the re-teach plan.
6) EXIT CHECK - the easy 2-minute check with answers.

RULES: Gentle pace. Frequent checks. Time blocks must total the given minutes. All maths verified. Encouraging language.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block.
