One-Prompt Deck for Gamma, Canva or Tome (AI Deck Builders)

A single clean, structured outline written so an AI deck builder turns it into a finished designed presentation in one go.

Makes images - needs an image AI  ·  Best tool:  Gamma / Canva Docs-to-Deck / Tome (free tier)

✅ How to use this effectively

  1. 1. In any AI chat, paste this prompt.
  2. 2. Enter topic, class and slide count.
  3. 3. Copy the structured outline it returns.
  4. 4. Open Gamma or Canva 'text to deck'.
  5. 5. Paste the outline and generate.
🔧 If it is not right, reply with this: If the builder makes too many or merged slides, reply: 'Reformat so each '# Slide' heading is one slide only, exactly [N] slides, nothing merged.'
THE PROMPT
ROLE: Act as an expert mathematics teacher and a prompt specialist who writes outlines optimised for AI deck builders such as Gamma, Canva and Tome.

CONTEXT: I teach [CLASS/GRADE] mathematics for [BOARD or EXAM]. I will paste your output into an AI presentation tool that auto-designs slides from a text outline. The output must be structured so the tool splits it into clean slides correctly and chooses good layouts and visuals.

YOUR TASK:
1. Read the topic I give you.
2. Produce a structured outline of exactly [N] slides (if I do not give N, use 12).
3. Use this strict per-slide structure so the builder parses it cleanly:
   # Slide [n]: [Title]
   - 3 to 5 concise bullets (max 9 words each)
   - Layout hint: [e.g. two-column, big-number, comparison, image-left]
   - Image idea: [a short description the tool can generate or search]
   - Notes: one line of speaker notes
4. Include a strong title slide, an agenda slide, concept slides, at least 2 worked-example slides, a common-mistakes slide, a summary slide and a final 'practice / exit question' slide.
5. Keep maths short and slide-friendly; spell out symbols where a design tool might mangle them (write 'x squared' as well as x^2 if helpful).

INPUT: Here is the topic and any must-cover sub-points: [PASTE THE TOPIC HERE].

HOW TO WORK IT OUT: Plan the slide split silently so each '# Slide' is genuinely one slide of content. Solve any worked example fully and verify each final answer by at least two independent checks before adding it. Never invent formulas, notation or values; use only standard results for this board. Keep all mathematics legible. Before finishing, confirm independently that (a) there are exactly [N] '# Slide' headings, and (b) every formula and answer is correct. Fix anything wrong before showing me.

OUTPUT FORMAT:
- First line only: a one-sentence 'Paste this whole block into Gamma or Canva text-to-deck' instruction.
- Then the structured outline using the exact '# Slide' format above, in order, nothing else between slides.
- Last line: a footer instruction the design tool can keep - 'Footer on every slide: Prepared by Indrajeet Yadav' - and put 'Compiled by Indrajeet Yadav' inside the title slide bullets.

RULES: Stick to the exact heading format so the builder parses it. No long paragraphs. No filler slides. Use language a [CLASS/GRADE] student understands. Image ideas must be relevant to the maths, not decorative clutter.

If anything is unclear, ask me one short question before you start.
