Attendance and Marks Tracker Template

A ready spreadsheet layout for attendance and marks with auto-total formulas.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt below.
  3. 3. Enter number of students and tests.
  4. 4. Send and read the column plan.
  5. 5. Copy the formulas into your sheet.
🔧 If it is not right, reply with this: If a formula does not work, reply: "The formula in column [LETTER] gives an error in Google Sheets. Rewrite it for Google Sheets and explain which cell to put it in."
THE PROMPT
ROLE: Act as a spreadsheet expert who builds simple trackers for teachers who are not technical.

CONTEXT: I want one tracker to record both attendance and test marks for my [CLASS/GRADE] [SUBJECT] class. I have about [NUMBER] students and [NUMBER] tests/assessments this term. I will use [Microsoft Excel / Google Sheets - pick one]. I am not technical, so explain where each formula goes.

YOUR TASK:
1. Design the column layout for an attendance sheet and a marks sheet (two sections or two tabs).
2. For attendance: columns for student name, roll no., then one column per day with P/A, plus a Total Present and Attendance % column.
3. For marks: columns for each test with its max marks, a Total, a Percentage, and a Grade.
4. Give the EXACT formula for each calculated column, written for my chosen app.
5. Tell me in plain words which cell to type each formula into and how to drag it down.

INPUT: Number of students: [NUMBER]. Tests and their max marks: [LIST, e.g. Test 1 = 20, Test 2 = 25, Final = 50]. Grade bands I use: [LIST, e.g. A = 90+, B = 75-89, or write "you suggest"].

HOW TO WORK IT OUT: Build the formulas correctly for the exact app I chose (Excel and Google Sheets differ). Mentally test each formula with one sample student to confirm the totals, percentage and grade come out right - show that one worked example. Recompute the sample percentage twice to be sure. Never use a function the chosen app does not support.

OUTPUT FORMAT (use this exact order):
1. ATTENDANCE LAYOUT: column list with letters (A, B, C...) and what goes in each.
2. ATTENDANCE FORMULAS: exact formula + the cell to put it in + plain instruction.
3. MARKS LAYOUT: column list with letters.
4. MARKS FORMULAS: exact formula + cell + instruction (Total, %, Grade).
5. WORKED EXAMPLE: one sample student row with the numbers filled in.
6. SETUP STEPS: 4 simple steps to start using it.

RULES: Formulas must match my chosen app. Explain like I have never used formulas before. Verify with the worked example.


MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.
At the very end add a footer line: "Prepared by Indrajeet Yadav", and write "Compiled by Indrajeet Yadav" as a suggested title for the top row of the sheet.
