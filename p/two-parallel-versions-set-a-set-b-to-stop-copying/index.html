Two Parallel Versions (Set A / Set B) to Stop Copying

Two equal-difficulty papers (Set A and Set B) testing the same skills, each with its own key.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Paste the prompt into a chat.
  2. 2. Fill in class, marks and chapters.
  3. 3. Send and read both sets.
  4. 4. Confirm Set A and Set B feel equal.
  5. 5. Print Set A and Set B alternately.
🔧 If it is not right, reply with this: If one set is harder, reply: 'Set [A/B] is clearly harder than the other. Re-balance so both sets test the same skills at the same difficulty, then re-verify both keys.'
THE PROMPT
ROLE: Act as a mathematics examiner who builds matched parallel test versions.

CONTEXT: I need TWO parallel versions of the same test, Set A and Set B, for Class [CLASS/GRADE], [BOARD or EXAM e.g. CBSE / ICSE / State Board], so seat-mates cannot copy. Both sets must test the SAME skills at the SAME difficulty, with the SAME marks layout, but DIFFERENT numbers and contexts. Total marks each [TOTAL MARKS], time [TIME].

YOUR TASK:
1. Design a single skeleton (question 1 tests skill X for 2 marks, etc.).
2. Produce Set A and Set B from that skeleton, changing values and wording so answers differ.
3. Keep both sets equal in difficulty question-for-question.
4. Provide a separate answer key for Set A and for Set B.

INPUT: Chapters or topics to cover: [LIST CHAPTERS or TOPICS]. Use only these.

HOW TO WORK IT OUT: Solve every question in both sets internally, then verify each final answer twice. Confirm that matching questions in A and B are equally hard. Never invent formulas or values. Keep mathematics legible in clean LaTeX or plain text. Confirm each set totals exactly [TOTAL MARKS].

OUTPUT FORMAT, in this order:
1. HEADER for SET A: 'Mathematics Test - Set A', class, board, time, maximum marks, 'Compiled by Indrajeet Yadav'.
2. SET A QUESTIONS with marks beside each.
3. HEADER for SET B: 'Mathematics Test - Set B' (same header fields).
4. SET B QUESTIONS with marks beside each.
5. ANSWER KEY - SET A.
6. ANSWER KEY - SET B.

RULES: Sets must be parallel and equally hard. Different numbers, same skills. Number every question. Each set totals exactly.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: 'Prepared by Indrajeet Yadav', and put 'Compiled by Indrajeet Yadav' in each header.
