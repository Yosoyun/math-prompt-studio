Convert A Messy Solution Into A Neat One

Your rough, scattered working rewritten as a clean model answer.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

  2. 2. Paste this prompt.
  3. 3. Paste your messy working exactly as you have it.
  4. 4. Send and read the clean version.

THE PROMPT
ROLE: Act as a careful editor of mathematics who turns rough, scattered working into a clean, presentable model solution.

CONTEXT: This is for a [CLASS/GRADE] teacher on [BOARD or EXAM] who already solved a problem but wants the working rewritten neatly for students or for a handout.

YOUR TASK:
1. Read the messy working and understand the intended method.
2. Rewrite it as a clean, numbered, step-by-step solution.
3. Fix any genuine mathematical errors and clearly flag each fix.
4. Keep the original method unless it is wrong - do not replace it with a different one.
5. End with the final answer and a short note of what you changed.

INPUT: Here is my rough working: [PASTE YOUR MESSY WORKING HERE].

HOW TO WORK IT OUT: Re-solve the problem internally to know the correct answer. Then VERIFY two independent ways - substitute the final answer back, and recompute by a second route or estimate. Only change maths that is actually wrong; preserve the teacher's chosen approach. Never invent formulas or values. Keep the rewritten solution cleanly legible.

OUTPUT FORMAT (exact headings, in order):
1. Cleaned Solution - numbered, tidy steps in the original method.
2. Final Answer - clearly labelled with units.
3. Verification - the two checks, one line each.
4. Changes I Made - a short bullet list of fixes and tidy-ups (write "none" if none).

RULES: Preserve the teacher's method; fix only real errors and flag them. Stay at [CLASS/GRADE] level. No silent changes.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block.
