Group Activity for One Class Period

A timed, structured group task with roles, rules and a quick assessment.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Fill in topic, class and group size.
  2. 2. Paste the prompt into any AI chat.
  3. 3. Read the minute-by-minute plan.
  4. 4. Assign the listed roles to students.
  5. 5. Use the quick check at the end.
🔧 If it is not right, reply with this: If the timing does not fit, reply: "My period is only 35 minutes. Re-time every stage so the total is 35 minutes and keep the same activity."
THE PROMPT
ROLE: Act as a maths teacher who runs lively, well-organised group activities that keep every student involved.

CONTEXT: I teach [CLASS/GRADE] under [BOARD e.g. CBSE / ICSE / State Board]. I want one GROUP ACTIVITY on [TOPIC e.g. fractions / linear equations / probability] for groups of [GROUP SIZE e.g. 4]. It must fit a [PERIOD LENGTH e.g. 40] minute period and keep all students active, not just the strongest.

YOUR TASK:
1. Design one collaborative activity (a challenge, a game, or a build) on the topic.
2. Give each group member a named role with a clear job.
3. Provide a minute-by-minute timeline that adds up to the period length.
4. State the rules clearly so the class runs smoothly.
5. Include the actual maths content groups will work on (questions, cards, or data).
6. Add a fast 5-mark assessment to grade each group.

INPUT: Here is my topic and class size: [PASTE TOPIC AND CLASS DETAILS HERE]. If blank, choose a strong activity for the class and board above.

HOW TO WORK IT OUT: Plan internally and make sure the timeline truly adds up to the period length; list each stage's minutes and show the total. Ensure every maths question or value is correct, and VERIFY any answers by a second method. Never invent formulas. Keep all maths legible in clean plain text or simple LaTeX.

OUTPUT FORMAT (use these exact headings in this order):
1. ACTIVITY TITLE
2. GOAL (one line)
3. GROUP ROLES (role name + job)
4. MATERIALS NEEDED
5. MINUTE-BY-MINUTE TIMELINE (stage | minutes; final Total)
6. THE ACTIVITY CONTENT (the questions/cards/data, with answers in a separate ANSWER KEY block)
7. RULES FOR THE CLASS
8. QUICK ASSESSMENT (5 marks, what earns each mark)
9. TEACHER TIPS

RULES: Every student must have something to do at all times. Be concrete; no vague 'discuss'. Keep language short and friendly.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title block at the top.
