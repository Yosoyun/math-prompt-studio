Friendly Homework Reminder for the Class Group

A clear, polite homework reminder ready for WhatsApp or email.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt with task and date.
  3. 3. Pick group or individual version.
  4. 4. Copy into your class group.
  5. 5. Reuse the format weekly.
🔧 If it is not right, reply with this: If it is too long for a group chat, reply: "Make it a 4-line WhatsApp message with the due date in bold words."
THE PROMPT
ROLE: Act as an organised maths teacher writing a clear homework reminder.

CONTEXT: I teach [SUBJECT] to [CLASS/GRADE] following [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. I need to remind students (and sometimes parents) about homework in a way that is friendly, specific, and impossible to misunderstand, suitable for a WhatsApp class group or email.

YOUR TASK:
1. Write a short reminder that states exactly what to do, from where, and by when.
2. Make the deadline and submission method unmissable.
3. Add one encouraging line so it motivates rather than nags.
4. Provide a GROUP version (to all) and a GENTLE INDIVIDUAL version (for a student who often forgets).

INPUT - the homework details (use only these facts):
- Task: [e.g. Exercise 5.2 Q1 to Q10 / worksheet on quadratics]
- Source/page/chapter: [DETAILS]
- Due date and time: [DATE, TIME]
- How to submit: [notebook in class / photo on group / portal]
- Anything special: [bring instruments / show full steps, optional]

HOW TO WORK IT OUT: Use only the facts above; do not invent exercise numbers, pages, or dates. If a detail is missing, leave a clear [placeholder]. Keep it skimmable on a phone screen. Re-read and ensure the WHAT, WHERE, and WHEN are all present and unambiguous.

OUTPUT FORMAT (use these exact headings):
GROUP REMINDER (WHATSAPP): 4 to 6 short lines, with the due date highlighted in CAPS or *stars*.
GROUP REMINDER (EMAIL): a slightly fuller version with a subject line.
GENTLE INDIVIDUAL NUDGE: 2 to 3 friendly lines for one student.

RULES:
- Crystal clear on deadline and submission; friendly, never scolding.
- No jargon; easy to read fast.
- Plain, clean text only.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the email header block.
