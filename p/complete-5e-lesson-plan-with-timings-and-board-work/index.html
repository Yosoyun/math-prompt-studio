Complete 5E Lesson Plan with Timings and Board-Work

A full 5E lesson plan: timed Engage-Explore-Explain-Elaborate-Evaluate, board-work plan, and assessment.

Works on any free AI  ·  Best tool:  ChatGPT or Claude (free)

✅ How to use this effectively

  1. Open ChatGPT or Claude in your browser.
  2. Fill [CLASS], [TOPIC], [BOARD] and minutes.
  3. Paste the whole prompt and send.
  4. If it stops mid-way, type 'continue'.
  5. Copy into Word, set A4, then print.
🔧 If it is not right, reply with this: If timings do not add up to your period, reply: 'Re-balance every stage so the timings sum exactly to [N] minutes, and show a running clock column.'
THE PROMPT
ROLE: Act as a master mathematics teacher and instructional designer who writes classroom-ready 5E lesson plans.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE foundation]. The topic is [TOPIC e.g. Quadratic Equations - nature of roots]. My single period is [N e.g. 45] minutes. Class strength is about [STRENGTH]. Available aids: blackboard/whiteboard only unless I state otherwise: [LIST ANY AIDS].

YOUR TASK:
1. Write a complete 5E lesson plan (Engage, Explore, Explain, Elaborate, Evaluate) with a minute-by-minute running clock so the stages sum exactly to [N] minutes.
2. State 2-3 measurable learning objectives in 'Students will be able to...' form, plus the prior knowledge assumed.
3. Give a concrete teacher script for each stage: the exact question I ask to Engage, the guided task for Explore, the worked example for Explain, a harder transfer task for Elaborate, and an Evaluate check.
4. Produce a BOARD-WORK PLAN: what to write on the left/centre/right of the board, in what order, with the key steps and the one box-it-up result.
5. List 3 likely misconceptions and the exact correction line for each.
6. End with an EXIT TICKET (2 questions) and a 3-question homework set with answers.

INPUT: Here is the exact topic and any sub-points to cover: [PASTE TOPIC / SYLLABUS LINE]. If I attach a textbook page, build the plan around it.

HOW TO WORK IT OUT: Solve every example and exit-ticket question fully in your head first and VERIFY each final answer by a second independent method (re-derive or substitute back). Never invent formulas or values. Keep all mathematics fully legible in plain text (use ^ for powers, sqrt() for roots, fractions as a/b). Confirm the stage minutes total exactly [N] before you output.

OUTPUT FORMAT (use these headings, in this order):
1. HEADER: Topic, Class, Board, Duration, Date line.
2. OBJECTIVES and PRIOR KNOWLEDGE.
3. MATERIALS.
4. 5E PLAN as a table: Stage | Time (running clock) | Teacher does | Students do | Key question.
5. BOARD-WORK PLAN (left/centre/right layout).
6. MISCONCEPTIONS and CORRECTIONS (table).
7. EXIT TICKET (2 Qs with answers).
8. HOMEWORK (3 Qs with answers and one-line method).

RULES: Keep it to one period. Plain ASCII maths. Realistic timings. No filler. Differentiate with one 'support' and one 'stretch' note per stage where natural.


OUTPUT FILE FORMAT: present the complete result as a clean, ready-to-use document I can paste straight into Microsoft Word or Google Docs - a clear bold title, section headings, neatly numbered items, comfortable spacing, and simple tables only where helpful (avoid heavy formatting that breaks on paste). It must also print cleanly to PDF on A4 (File > Print > Save as PDF). Keep all mathematics fully readable (fractions as a/b, powers as x^2, roots as sqrt(x)).

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.
At the very end add a footer line: 'Prepared by Indrajeet Yadav', and put 'Compiled by Indrajeet Yadav' in the header block at the top.
