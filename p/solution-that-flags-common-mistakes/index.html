Solution That Flags Common Mistakes

The correct solution plus the exact traps students fall into.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste this prompt.
  3. 3. Add your question and the class level.
  4. 4. Send and read the solution and the trap list.
  5. 5. Warn students about each trap in class.
🔧 If it is not right, reply with this: If the warnings are generic, reply: "Make the mistake list specific to THIS problem's numbers and steps."
THE PROMPT
ROLE: Act as an experienced classroom teacher who knows exactly where students go wrong and points it out while solving.

CONTEXT: This is for [CLASS/GRADE] students on [BOARD or EXAM]. The goal is a correct solution AND a clear warning at each spot where students typically slip.

YOUR TASK:
1. Solve the problem correctly, step by step.
2. At each risky step, add a short Mistake Alert describing the specific wrong move students make and the right move instead.
3. After the solution, list the top mistakes for this exact problem.
4. State the final answer with units.
5. Add one line on the habit that prevents these errors.

INPUT: Here is the question: [PASTE THE QUESTION HERE].

HOW TO WORK IT OUT: Solve internally first. Then VERIFY the answer two independent ways - substitute back, and use a second route or estimate. Make the mistake alerts concrete to this problem's actual numbers and steps, not vague advice. Never invent formulas or values; use only correct, level-appropriate maths. Keep everything cleanly legible.

OUTPUT FORMAT (exact headings, in order):
1. Correct Solution - numbered steps; insert "Mistake Alert:" lines right where the trap is.
2. Final Answer - clearly labelled with units.
3. Top Mistakes For This Problem - a short bullet list, each one specific.
4. Verification - the two checks, one line each.
5. Prevention Habit - one line.

RULES: Mistake alerts must be specific and real, tied to this problem. Stay at [CLASS/GRADE] level. Solution itself must be flawless.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block.
