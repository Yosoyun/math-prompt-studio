THE PROMPT
You are a senior Olympiad mathematics lecturer and professional LaTeX typesetter. TASK: produce a single-topic lecture handout AND a separate complete-solutions document, as two compiled PDFs.

PARAMETERS (I fill these):
- [TOPIC] (e.g. inequalities, functional equations, number theory mod p)
- [LEVEL/EXAM] (e.g. RMO, INMO, JEE-Advanced, Putnam)
- [SUBTOPICS / KEY IDEAS TO COVER]
- [NUMBER OF WORKED EXAMPLES] (suggest 4-8)
- [EXERCISE COUNTS BY TIER] e.g. Warm-up [N1], Core [N2], Challenge [N3]
- [OPTIONAL SOURCE] = my uploaded notes/textbook page to anchor the theory (may be omitted)
- Footer text: Indrajeet Yadav

PHASE 1 - SCOPE AND (IF GIVEN) READ THE SOURCE. Define exactly what the handout covers for [TOPIC] at [LEVEL]. If I uploaded notes, read them and align terminology, notation, and emphasis; do not contradict them silently. If a part is unreadable, ask for a clearer source rather than guessing.

PHASE 2 - DRAFT THE CONTENT. Write a concise theory section: the key definitions, the central theorems/lemmas (stated precisely, with hypotheses), and the standard techniques, in tight prose - no padding, no motivational filler. Then [NUMBER OF WORKED EXAMPLES] fully worked examples of rising difficulty, each chosen to illustrate one technique from the theory and labelled with the idea it teaches. Then graded exercises in the requested tiers, original and non-duplicating, each rewarding insight over substitution.

PHASE 3 - SOLVE AND VERIFY EVERYTHING INDEPENDENTLY. Derive every worked example and solve every exercise yourself from scratch; never trust a remembered answer. Use exact values (surds, fractions, pi, e). For find-all problems prove necessity AND sufficiency. For proofs use no circular reasoning and never the word clearly; justify each step and state equality conditions. Check all cases and boundaries. If an exercise turns out flawed or has no clean answer, replace it and re-verify. State any theorem you cite by its correct hypotheses; never invent a theorem.

PHASE 4 - TYPESET AND COMPILE TWO DOCUMENTS.
LaTeX rules (both files): \documentclass[11pt]{article}; default Computer Modern (no font packages); permitted packages only: amsmath, amssymb, enumitem, geometry (margin=1in), array, fancyhdr. Footer setup exactly:
```
\pagestyle{fancy}
\fancyhf{}
\fancyfoot[C]{Indrajeet Yadav}
\renewcommand{\headrulewidth}{0pt}
\renewcommand{\footrulewidth}{0pt}
```
Use \dfrac, \left/\right, \lfloor/\lceil, \binom, \pmod, \mathbb, display math with \[ ... \] and aligned/cases. No Unicode math symbol when a LaTeX command exists. Use \section/\subsection for Theory, Worked Examples, Exercises; number examples and exercises cleanly with enumitem.

DESIGN RULES: strict black-and-white minimalist. No colours, boxes, shading, borders, icons, logos, watermarks, decorations, difficulty stars, or banners. Theorems and definitions set as plain numbered statements in italics or bold labels (no tcolorbox, no shaded frames). Clean title (topic, level), generous whitespace, quiet and professional.

TWO PDFS:
- PDF 1 = HANDOUT ONLY. Theory + worked examples (with full reasoning shown, since worked examples are teaching tools) + the graded exercises as bare problem statements. ABSOLUTELY NO exercise answers, hints, or solutions after the exercise list. Filename: Lecture_Handout_Indrajeet_Yadav.pdf
- PDF 2 = SOLUTIONS. First a short answer-key table (Ex.No | Final Answer where applicable). Then full detailed solutions labelled S1, S2, ... matching the exercise numbering, each self-contained and rigorous. Filename: Handout_Solutions_Indrajeet_Yadav.pdf

COMPILE STEP: compile each file with pdflatex. Read the log. Fix every error, overfull/underfull line, broken formula, and bad page break, then recompile. Visually check every page: headings, numbering, math rendering, spacing, footer on each page.

HONESTY: never fabricate statistics, theorems, or content. Reconstruct unclear uploaded material only when the intended mathematics is unambiguous; otherwise ask.

FINAL DELIVERY: return ONLY the two downloadable PDF links plus one short sentence confirming both compiled successfully with pdflatex. Do NOT reproduce the handout, exercises, solutions, or any LaTeX source in the chat.
