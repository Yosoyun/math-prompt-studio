Match-the-Column Worksheet (Link Concepts Fast)

A two-column matching exercise (and a bonus second set) with a clean matching key.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Name the topic to match on.
  2. 2. Say what Column A and B hold.
  3. 3. Add class and board.
  4. 4. Send and check the matches.
  5. 5. Print the activity.
🔧 If it is not right, reply with this: If two options match the same item, reply: "Some items have more than one correct match. Make each match unique (one-to-one) and re-verify the key."
THE PROMPT
ROLE: Act as a mathematics teacher who designs match-the-column activities that test understanding quickly.

CONTEXT: I teach [CLASS/GRADE] under [BOARD or EXAM e.g. CBSE / ICSE / State Board]. I want a match-the-column worksheet on [TOPIC] to check whether students can connect related ideas (for example, a function with its derivative, a problem with its answer, a term with its definition, a shape with its formula).

YOUR TASK:
1. Build Set 1: Column A with 8 items and Column B with 8 items, where each A matches exactly ONE B (strict one-to-one).
2. Decide what A and B hold based on what I specify, e.g. A = expressions, B = simplified results.
3. Add 2 'distractor' extra options in Column B (so 8 in A, 10 in B) to make guessing harder. Clearly say there are 2 extras that match nothing.
4. Build Set 2: a second, slightly harder matching set of 6 pairs on the same topic.
5. Show how many marks each correct match is worth and the total.

INPUT: Here is the topic and exactly what Column A and Column B should contain: [PASTE TOPIC AND COLUMN CONTENTS HERE].

HOW TO WORK IT OUT: Work out every correct pairing internally first. VERIFY each pairing two independent ways and confirm no item in A matches more than one B. Never invent values. Keep all maths fully legible in clean LaTeX or clear plain text.

OUTPUT FORMAT (exact headings, in order):
- TITLE BLOCK: 'Match the Column', Topic, Class, Total Marks, and 'Compiled by Indrajeet Yadav'.
- SET 1: Column A (1-8) and Column B (a-j) side by side, with note about 2 extra distractors.
- SET 2: the harder 6-pair set.
- ANSWER KEY: matching pairs for both sets, e.g. 1-c, 2-f, ... clearly separated at the end.

RULES: Strict one-to-one matching. Distractors must be plausible but wrong. Key must match. Simple, clear wording.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.
