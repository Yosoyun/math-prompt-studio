Printable Snakes-and-Ladders Maths Board (Image)

A colourful A4 board game image with a numbered path, challenge squares and a plain-text answer key below.

THE PROMPT
ROLE: Act as a graphic designer and mathematics teacher who must DRAW a finished, colourful printable game board as an image.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. I want a single-page snakes-and-ladders style maths board game image on [TOPIC] that I can print on A4 and play in class with dice and tokens.

YOUR TASK:
1. Design ONE attractive A4 portrait board with a numbered path of about 30 squares, snakes and ladders, a START and a FINISH.
2. Place short maths challenges from [TOPIC] on at least 12 squares (write the problem text legibly inside or beside those squares).
3. Add a small rules box and a 'challenge card' legend in a corner.
4. Keep colours bright, lines clean, and ALL text crisp and perfectly spelled.

INPUT: Topic, class and board are above. If I add nothing, choose challenges squarely on-topic and suitable for the class.

HOW TO WORK IT OUT: Before drawing, solve every square's maths challenge and verify each answer twice by independent methods so nothing printed is wrong. Render the board so every number and every problem is sharp and readable; if any text would be cramped, enlarge that square. Never invent formulas or values. Provide a separate plain-text ANSWER KEY below the image.

OUTPUT FORMAT:
- First, the finished A4 board game as a single printable image (numbered path, snakes, ladders, challenge squares with legible problems, rules box, legend).
- Then below the image, a plain-text ANSWER KEY listing each challenge square number and its answer in a box [Answer: ...].

RULES: This output is an IMAGE, so use an image-capable AI. Every printed problem must be verified and correct. All text must be legible and correctly spelled. The path must be continuous from START to FINISH with no broken squares.

SIGNATURE: Sign the bottom-right of the page with an elegant handwritten signature "Indrajeet Yadav" with a small underline flourish.
