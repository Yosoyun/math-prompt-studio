Printable A4 Formula Sheet (Ready to Print)

A neatly laid-out, print-ready A4 formula sheet with clean spacing.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Paste the prompt into the chat.
  2. 2. Enter chapter, class, and board.
  3. 3. Send and copy the result.
  4. 4. Paste into Word or Google Docs.
  5. 5. Print on A4 and check it fits.
🔧 If it is not right, reply with this: If it overflows the page, reply: "This spills onto a second page. Trim and re-balance the two columns so it fits one A4 sheet, and verify the formulas."
THE PROMPT
ROLE: Act as a mathematics teacher and a careful document formatter.

CONTEXT: I teach [CLASS/GRADE] for the [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. I need a print-ready A4 formula sheet for [CHAPTER NAME] that I can paste straight into Word or Google Docs and print on ONE A4 page with neat two-column spacing.

YOUR TASK:
1. List the chapter's key formulas, grouped under short sub-headings.
2. Lay them out to suggest a balanced TWO-COLUMN A4 layout (tell me which formulas go in the left column and which in the right).
3. Keep total content to what genuinely fits one A4 page at a readable font.
4. Use clean, copy-friendly formatting (plain text or simple markdown) that survives pasting into a document.
5. Suggest a font size and margins so it prints well.

INPUT: Here is the chapter and class. Chapter: [PASTE HERE]. Class/Board: [PASTE HERE].

HOW TO WORK IT OUT: Internally confirm each formula is correct, then estimate length to ensure it fits one A4 page; trim the least important items if it would overflow. VERIFY two ways: (a) re-check each formula for accuracy; (b) re-check the amount of content is realistic for a single readable A4 page. Never invent formulas. Keep all maths fully legible in clean plain text or simple LaTeX so it pastes cleanly.

OUTPUT FORMAT (use this exact order):
1. TITLE: '[Chapter] - Formula Sheet (Class [X], [Board])'.
2. 'Compiled by Indrajeet Yadav'.
3. 'LEFT COLUMN:' with its grouped formulas, then 'RIGHT COLUMN:' with its grouped formulas.
4. A short 'Print settings' note (suggested font size, margins, page = A4).
5. Footer line.

RULES: Must fit one A4 page. Balanced columns. Copy-friendly formatting. Accurate formulas only.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE (mandatory): At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block.
