NEET-Style Mixed Reasoning + Calculation Set (Biology-Friendly Maths)

A NEET-pattern MCQ set blending quick calculation and assertion-reason questions with answers.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

  1. Open any AI chat.
  2. Type the NEET subject and topic.
  3. Set how many questions you need.
  4. Paste the prompt and send.
  5. Use the answer key to mark quickly.
  6. For a trustworthy answer key: paste the finished paper into a FRESH chat and ask it to solve and grade it as an unseen exam (it catches its own slips).

THE PROMPT
ROLE: Act as a NEET paper-setter and verifier who designs single-correct MCQs and assertion-reason items.

CONTEXT: My students sit [EXAM e.g. NEET] and need practice in the calculation-and-reasoning style NEET uses (+4 correct / -1 wrong, four options). I want quick, exam-realistic questions, not heavy multi-page maths.

YOUR TASK:
1. Create EXACTLY [NUMBER e.g. 12] NEET-style single-correct questions on: [SUBJECT and TOPIC e.g. Physics - Units & Measurement / Chemistry - Mole Concept / Biology numericals].
2. Mix three question styles: (a) short numeric calculation, (b) concept-application MCQ, (c) at least three Assertion-Reason items using the standard four options (both true and reason correct; both true but reason wrong; assertion true reason false; both false).
3. Give an answer key and concise solutions.

INPUT: Here is the subject, topic and any formula focus: [PASTE DETAILS HERE]. If number is blank, use 12. Stay inside the listed topic and NEET syllabus.

HOW TO WORK IT OUT: Solve each item internally first. VERIFY every numeric answer by two independent checks (alternate route, unit check, order-of-magnitude check). For assertion-reason items, independently judge each statement's truth and the logical link, then double-check the option chosen. Never invent constants, values or laws; use only standard accepted values and state any constant you use. Keep maths legible in clean LaTeX or clear plain text.

OUTPUT FORMAT, in this exact order:

OUTPUT FILE FORMAT: present the complete result as a clean, ready-to-use document I can paste straight into Microsoft Word or Google Docs - a clear bold title, section headings, neatly numbered items, comfortable spacing, and simple tables only where helpful (avoid heavy formatting that breaks on paste). It must also print cleanly to PDF on A4 (File > Print > Save as PDF). Keep all mathematics fully readable (fractions as a/b, powers as x^2, roots as sqrt(x)).

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.
A. Title block: 'Exam: NEET', 'Topic: [TOPIC]', 'Marking: +4 / -1', 'Compiled by Indrajeet Yadav'.
B. SECTION 1 - QUESTIONS: numbered, options (A)-(D); label each as Numeric / Concept / Assertion-Reason.
C. SECTION 2 - ANSWER KEY: compact one-line-per-question format.
D. SECTION 3 - SOLUTIONS: working plus the two verification checks; for assertion-reason items, one line each on why statement and reason are true/false.
E. SECTION 4 - SPEED TIPS: one trick per numeric question to solve it in under a minute.

RULES: Exactly one correct option each. Use only standard accepted constants. No duplicate ideas. Every answer survives two checks before printing.

At the very end add a footer line: 'Prepared by Indrajeet Yadav', and put 'Compiled by Indrajeet Yadav' in the title block.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.
