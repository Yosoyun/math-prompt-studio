Venn and Set Diagram with Shaded Regions

A correct two- or three-set Venn diagram with the right region shaded and elements placed.

Makes images - needs an image AI  ·  Best tool:  ChatGPT or Gemini (image)

THE PROMPT
ROLE: Act as a set-theory teacher who draws accurate Venn diagrams with the correct regions shaded.

CONTEXT: I teach [CLASS/GRADE] under [BOARD or EXAM e.g. CBSE / ICSE / JEE]. The shaded region must exactly match the set expression, which students often get wrong.

YOUR TASK:
1. Draw a rectangle for the universal set U, labelled in a corner.
2. Draw two or three overlapping circles for the sets, each clearly labelled (A, B, C).
3. Shade exactly the region described by the set expression I give, and leave everything else unshaded.
4. If I provide elements or counts, place each element or number in its correct region.
5. Keep circles cleanly overlapping so every sub-region is visible.

INPUT: Here is what I need: Sets and any elements: [e.g. A = {1,2,3,4}, B = {3,4,5,6}, U = {1..8}]. The expression to shade: [e.g. A union B, A intersection B, A minus B, (A union B)', A intersection B intersection C].

HOW TO WORK IT OUT: First write, in words, exactly which sub-regions the expression includes (for example, "A minus B means the part of A that is not in B: shade only the left crescent"). If elements or counts are given, compute which region each belongs to and show the working. VERIFY two ways: confirm the shaded sub-regions match your words, and, if counts are given, confirm all region counts add up to the size of U. Never shade an extra region; never misplace an element.

OUTPUT FORMAT:
1. REGION DESCRIPTION: plain words naming each sub-region the expression covers.
2. PLACEMENT (if elements given): which element or count goes in which region.
3. THE DIAGRAM: one clean labelled Venn image with the correct region shaded.
4. CHECK: one line confirming counts add to U (or "No counts given").

RULES: Only the described region is shaded. U shown as a labelled rectangle. Every circle and element labelled. Plain white background.

SIGNATURE: Sign the bottom-right of every page with an elegant handwritten signature "Indrajeet Yadav" with a small underline flourish.
