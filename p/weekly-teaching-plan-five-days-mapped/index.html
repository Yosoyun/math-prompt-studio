Weekly Teaching Plan, Five Days Mapped

A whole-week plan with daily objectives, content, practice and homework.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt and the week's topics.
  3. 3. Send and review the daily split.
  4. 4. Adjust any day that feels heavy.
  5. 5. Save or print the week plan.
🔧 If it is not right, reply with this: If one day is overloaded, reply: "Friday has too much. Rebalance the week so each day is similar in load, keeping all topics covered."
THE PROMPT
ROLE: Act as a mathematics teacher who plans a productive teaching week.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM]. I have [NUMBER] periods this week, each [LENGTH], spread across [NUMBER] days. I want a week plan that covers the listed topics, builds logically day to day, and includes practice, homework and one short review. Class ability is [weak / mixed / strong].

YOUR TASK:
1. Distribute the week's topics across the days in a sensible teaching order.
2. For each day, give the objective, what to teach, in-class practice, and homework.
3. Link each day to the next so learning builds.
4. Reserve time on the last day for review and a short check.
5. Flag any topic that may spill into next week.

INPUT: Here are this week's topics or syllabus points: [PASTE THE WEEK'S TOPICS HERE].

HOW TO WORK IT OUT: First count topics and periods, then distribute so they fit. VERIFY by adding the periods used and confirming they equal the periods available. Solve any sample questions internally and verify answers two ways. Do not invent syllabus beyond what I give plus standard prerequisites.

OUTPUT FORMAT (exact headings, in order):

OUTPUT FILE FORMAT: present the complete result as a clean, ready-to-use document I can paste straight into Microsoft Word or Google Docs - a clear bold title, section headings, neatly numbered items, comfortable spacing, and simple tables only where helpful (avoid heavy formatting that breaks on paste). It must also print cleanly to PDF on A4 (File > Print > Save as PDF). Keep all mathematics fully readable (fractions as a/b, powers as x^2, roots as sqrt(x)).

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.
1. TITLE BLOCK - week, class, board, total periods, 'Compiled by Indrajeet Yadav'.
2. WEEK OVERVIEW - one line per day.
3. DAILY PLAN TABLE - columns: Day | Objective | Teach | In-class practice | Homework.
4. DAY-TO-DAY LINKS - how each day connects to the next.
5. REVIEW AND CHECK - the last-day review with sample questions and answers.
6. SPILL-OVER FLAG - anything that may move to next week.
7. PERIOD COUNT CHECK - confirm periods used equals periods available.

RULES: Days must be balanced in load. Periods used must equal periods available. Order must be logical.

At the very end add a footer line: 'Prepared by Indrajeet Yadav', and put 'Compiled by Indrajeet Yadav' in the title block.
