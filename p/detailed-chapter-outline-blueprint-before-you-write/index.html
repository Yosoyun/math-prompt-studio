Detailed Chapter Outline / Blueprint Before You Write

A section-by-section skeleton of the chapter so writing it becomes easy.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Enter the topic, class and board.
  2. 2. Send the prompt and read the outline.
  3. 3. Add or remove sections you want.
  4. 4. Save it as your writing plan.
🔧 If it is not right, reply with this: If the order feels off, reply: "Reorder the sections so prerequisites come first, and tell me which earlier topic each section assumes."
THE PROMPT
ROLE: Act as a mathematics curriculum designer and textbook editor.

CONTEXT: I am about to write a textbook chapter on [TOPIC] for [CLASS/GRADE] students preparing for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. Before writing, I want a clear blueprint so nothing important is missed and the flow is logical.

YOUR TASK:
1. Produce a detailed chapter outline, not the full content.
2. List the prerequisite ideas a student must already know.
3. Break the chapter into 5 to 9 numbered sections in the best teaching order.
4. Under each section, give 2-4 bullet sub-points of what to cover.
5. Mark where solved examples should appear and roughly how many.
6. Suggest where to place special boxes (Key Point, Caution, Did You Know).
7. Recommend how many exercise questions per section and at what difficulty.

INPUT: Here is the syllabus scope or anything specific I must include: [PASTE HERE OR WRITE "use the standard syllabus"].

HOW TO WORK IT OUT: Think through the logical dependency order internally first. VERIFY the outline two ways: (a) check that every section's prerequisites appear earlier, and (b) check that the listed scope fully matches the stated syllabus with nothing missing or extra. Do not invent topics outside the syllabus. Keep any sample notation in clean, legible form.

OUTPUT FORMAT (exact order):
1. Chapter Title (proposed)
2. Prerequisites
3. Section-by-Section Outline (numbered sections with sub-bullets)
4. Suggested Solved-Example Placement
5. Suggested Special Boxes
6. Suggested Exercise Plan (per section, with difficulty mix)
7. Estimated Page or Word Count

RULES: Keep it skimmable. Use numbers and bullets, not long paragraphs. Logical order is the top priority. Flag any section likely to be hard for this level.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the header block at the top.
