Solve Every Question on the Page

Full solutions to all questions on the page, numbered in order, each verified.

Attach a photo of the question first  ·  Best tool:  ChatGPT or Gemini (image)

✅ How to use this effectively

  1. 1. Photograph the full page so all questions are sharp.
  2. 2. Attach the photo to the chat.
  3. 3. Paste this prompt and send.
  4. 4. Read solutions in number order.
  5. 5. If it stops early, reply "continue from the next question".
🔧 If it is not right, reply with this: If it skips a question, reply: "You missed question [number]. Solve it now with full steps and verification."
THE PROMPT
ROLE: Act as a thorough mathematics teacher who solves an entire page of questions accurately and in order.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET], topic [TOPIC]. My photo shows a page containing several questions. I want every one of them solved.

YOUR TASK:
1. Read the attached photo and list all question numbers you can see, in order.
2. Solve EACH question fully, in the same order, keeping them clearly separated.
3. If the page is long, solve as many as you can, then tell me where you stopped so I can ask you to continue.

INPUT: I have attached a photo of a page of questions. If any question is partly cut off or unreadable, label it clearly (for example "Q4 - partly unreadable"), state your best assumption, and solve what you can.

HOW TO WORK IT OUT: Solve each problem silently first. For every single question, VERIFY the final answer with at least TWO independent checks (substitute back, and re-derive or estimate). If a question's checks disagree, fix it before moving on. Never invent formulas or values; name standard results. Keep all mathematics fully legible in clean LaTeX or clear plain text, one step per line.

OUTPUT FORMAT: First a line "Questions found: ..." listing the numbers. Then, for EACH question, use this block exactly:
=== Q[number] ===
- Question: restated in clean text.
- Solution: numbered steps, one idea per line, short reasons.
- Final answer: clearly stated.
- Verification: Check 1 and Check 2, each in one line.
Leave a blank line between question blocks.

RULES: Do not merge two questions into one solution. Do not skip any visible question. If you run out of space, end with a line "STOPPED AFTER Q[number] - reply 'continue' for the rest." Show units where needed. Use plain ASCII only with straight quotes. Be exact unless an estimate is requested.


MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.
At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block at the top.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.
