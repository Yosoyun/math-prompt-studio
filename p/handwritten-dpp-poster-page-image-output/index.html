Handwritten DPP Poster Page (Image Output)

A single beautiful handwritten-style DPP page image you can post in class or share on WhatsApp.

Makes images - needs an image AI  ·  Best tool:  ChatGPT or Gemini (image mode)

✅ How to use this effectively

  1. 1. Open an image-capable AI (ChatGPT/Gemini).
  2. 2. Fill topic, class and count.
  3. 3. Paste the prompt and send.
  4. 4. Wait for the page image.
  5. 5. Download and share or print it.
🔧 If it is not right, reply with this: If any maths in the image looks wrong or messy, reply: "Re-make the page; question [number] has a maths error - fix it and keep the handwriting clean and readable."
THE PROMPT
ROLE: Act as a mathematics teacher AND a premium handwriting artist who designs attractive single-page practice posters.

CONTEXT: Create ONE portrait image that looks like a neat handwritten Daily Practice Problem page for [CLASS/GRADE] students of [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. Topic = [TOPIC]. Number of questions = [COUNT e.g. 6].

YOUR TASK:
1. Design a single clean page titled 'Daily Practice Problem - [TOPIC]'.
2. Write [COUNT] numbered questions in tidy, fully legible handwriting on lined or grid paper.
3. Put a small boxed ANSWER KEY at the bottom corner in a different ink colour.
4. Add light, tasteful decoration (a header banner, simple doodles) without crowding the maths.

INPUT: Topic = [TOPIC]. Class = [CLASS/GRADE]. Board/Exam = [BOARD or EXAM]. Count = [COUNT]. If unclear, assume and state it.

HOW TO WORK IT OUT: Before drawing, solve every question internally and verify each final answer by two independent checks so the printed answer key is correct. Never invent formulas or values. Every symbol, fraction, power and root in the image must be legible and mathematically correct - prefer fewer questions over cramped or wrong maths.

OUTPUT FORMAT:
- Exactly ONE portrait page image (not a collage, not multiple panels).
- Top: title banner with topic, class and board/exam.
- Middle: the [COUNT] numbered questions, neatly handwritten and well spaced.
- Bottom corner: a small boxed answer key in a second colour.
- The whole page must be readable at phone size.

RULES: One page only. Maths must be correct and legible. Decoration must never obscure any question. Keep colours calm and professional.

SIGNATURE: Sign the bottom-right of the page with an elegant handwritten signature 'Indrajeet Yadav' with a small underline flourish.
