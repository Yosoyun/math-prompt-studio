Class WhatsApp Announcement (Clear and Polite)

A short, professional announcement for the class or parent group.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt; state the news.
  3. 3. Add date, time, and any action needed.
  4. 4. Copy into the group.
  5. 5. Reuse the format for future notices.
🔧 If it is not right, reply with this: If it is too wordy, reply: "Cut to 4 lines max, put the date and action in bold, keep it polite."
THE PROMPT
ROLE: Act as a clear, courteous maths teacher writing a class or parent WhatsApp announcement.

CONTEXT: I run a class group for [CLASS/GRADE] studying [SUBJECT] for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. I need to post an announcement that is short, polite, easy to read on a phone, and leaves no room for confusion. The audience is [students / parents / both].

YOUR TASK:
1. Write a crisp announcement that states the news, the key date/time, and any action required.
2. Make the most important detail (date, deadline, or action) stand out.
3. Keep it warm and professional, not stiff or robotic.
4. Add a one-line variant in case it is going to PARENTS instead of students.

INPUT - the announcement details (use only these facts):
- What I am announcing: [e.g. test postponed / extra class / holiday / new timing / submit form]
- Key date and time: [DETAILS]
- Action required (if any): [e.g. carry calculator / reply YES / pay by date]
- Audience: [students / parents / both]
- Tone: [friendly / formal]

HOW TO WORK IT OUT: Use only the facts above; never invent dates, fees, or actions. If a detail is missing, insert a clear [placeholder]. Keep it scannable on a small screen. Re-read and confirm the WHAT, WHEN, and ACTION are all unmistakable.

OUTPUT FORMAT (use these exact headings):
ANNOUNCEMENT (WHATSAPP): 3 to 5 short lines, key detail in CAPS or *stars*.
PARENT VERSION: a slightly more formal one-paragraph variant with a polite greeting.
OPTIONAL SUBJECT LINE: in case I send it by email instead.

RULES:
- Short, polite, unambiguous; no jargon.
- Always include the date/time and any required action.
- Plain, clean text only.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.
