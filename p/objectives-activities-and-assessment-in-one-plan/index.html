Objectives, Activities and Assessment in One Plan

A tightly aligned plan where every objective links to an activity and a check.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt and your topic.
  3. 3. Send and read the alignment table.
  4. 4. Confirm each objective has a check.
  5. 5. Print the plan.
🔧 If it is not right, reply with this: If an objective has no matching check, reply: "Objective 2 has no assessment item. Add one assessment that directly tests it, with the answer."
THE PROMPT
ROLE: Act as a mathematics curriculum designer who insists on tight alignment.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM]. I want a lesson where every learning objective is matched to a teaching activity and to an assessment item, so nothing is taught that is not checked, and nothing is checked that was not taught. Lesson length is [DURATION].

YOUR TASK:
1. Write 3 to 4 measurable objectives using action verbs (define, compute, prove, apply).
2. For each objective design one classroom activity that builds it.
3. For each objective design one assessment item that tests it, with a full answer.
4. Build an alignment table linking objective to activity to assessment.
5. Add success criteria so I know when each objective is met.

INPUT: Here is the topic: [PASTE THE TOPIC HERE].

HOW TO WORK IT OUT: Solve every assessment item internally first. VERIFY each answer two independent ways and never invent formulas or data. Keep maths legible in clean plain text or simple LaTeX. Check that every objective appears in both the activity column and the assessment column.

OUTPUT FORMAT (exact headings, in order):

OUTPUT FILE FORMAT: present the complete result as a clean, ready-to-use document I can paste straight into Microsoft Word or Google Docs - a clear bold title, section headings, neatly numbered items, comfortable spacing, and simple tables only where helpful (avoid heavy formatting that breaks on paste). It must also print cleanly to PDF on A4 (File > Print > Save as PDF). Keep all mathematics fully readable (fractions as a/b, powers as x^2, roots as sqrt(x)).

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.
1. TITLE BLOCK - topic, class, board, duration, 'Compiled by Indrajeet Yadav'.
2. LEARNING OBJECTIVES - numbered, measurable.
3. SUCCESS CRITERIA - what a student can do if the objective is met.
4. ALIGNMENT TABLE - columns: Objective | Activity | Assessment item.
5. ACTIVITIES IN DETAIL - steps for each activity.
6. ASSESSMENT ITEMS WITH ANSWERS - each with full solution and the two checks.
7. ALIGNMENT CONFIRMATION - confirm every objective is both taught and tested.

RULES: Objectives must be measurable. Every objective must map to one activity and one assessment. All answers verified.

At the very end add a footer line: 'Prepared by Indrajeet Yadav', and put 'Compiled by Indrajeet Yadav' in the title block.
