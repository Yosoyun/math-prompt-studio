Clean Weekly Class Timetable

A tidy day-by-day weekly timetable grid with no clashes.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt below.
  3. 3. List your periods, days and subjects.
  4. 4. Send and check the grid.
  5. 5. Ask it to swap any period you dislike.
🔧 If it is not right, reply with this: If two periods clash or a free slot is missing, reply: "You scheduled [CLASS] twice on [DAY]. Fix all clashes and confirm I still have [NUMBER] free periods."
THE PROMPT
ROLE: Act as a school timetable coordinator who builds clash-free weekly schedules.

CONTEXT: I am a [SUBJECT] teacher. My school runs [NUMBER] periods a day, [NUMBER] days a week (Monday to [LAST DAY]). Each period is [NUMBER] minutes. There is a break/lunch after period [NUMBER].

YOUR TASK:
1. Build a weekly timetable grid for me using the classes and load I provide.
2. Spread each class's periods evenly across the week, not all on one day.
3. Keep my requested free periods empty.
4. Avoid putting the same class in the last period every day.
5. Leave the break row clearly marked.

INPUT: Here is my teaching load - paste each line as "Class - Subject - periods per week": [PASTE YOUR CLASSES AND PERIODS HERE, e.g. Class 9A - Maths - 6; Class 10B - Maths - 5]. My fixed free periods are: [LIST ANY, or write NONE]. Any fixed periods I cannot move: [LIST, or write NONE].

HOW TO WORK IT OUT: First add up the total periods I need per week and compare with total slots available (periods per day times days minus my free periods). If it does not fit, tell me before building. After building, scan every day and every class to confirm NO class appears twice in one day unless it must, and confirm the weekly count for each class matches my input exactly. Recount each class total twice. Never silently drop a period.

OUTPUT FORMAT (use this exact order):
1. FIT CHECK: total periods needed vs total slots available, one line.
2. WEEKLY GRID: a table with rows = period numbers (and a BREAK row) and columns = days. Each cell shows Class - Subject or FREE.
3. PER-CLASS COUNT: a short list confirming periods scheduled for each class.
4. NOTES: any compromise I should know about.

RULES: No class double-booked in the same slot. Keep free periods exactly as I asked. Plain, readable table.


OUTPUT FILE FORMAT: present the complete result as a clean, ready-to-use document I can paste straight into Microsoft Word or Google Docs - a clear bold title, section headings, neatly numbered items, comfortable spacing, and simple tables only where helpful (avoid heavy formatting that breaks on paste). It must also print cleanly to PDF on A4 (File > Print > Save as PDF). Keep all mathematics fully readable (fractions as a/b, powers as x^2, roots as sqrt(x)).

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.
At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the header block above the grid.
