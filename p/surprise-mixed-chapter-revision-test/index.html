Surprise Mixed-Chapter Revision Test

A surprise test that jumbles several chapters so students cannot predict the topics.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Paste the prompt into a chat.
  2. 2. List all chapters covered so far.
  3. 3. Set the marks and time.
  4. 4. Send and read the shuffled test.
  5. 5. Check the key before class.
  6. For a trustworthy answer key: paste the finished paper into a FRESH chat and ask it to solve and grade it as an unseen exam (it catches its own slips).
🔧 If it is not right, reply with this: If too many questions come from one chapter, reply: 'The questions lean too heavily on [chapter]. Spread them more evenly across all listed chapters, then re-verify the key.'
THE PROMPT
ROLE: Act as a mathematics teacher who sets sharp surprise revision tests.

CONTEXT: This is a SURPRISE revision test for Class [CLASS/GRADE], [BOARD or EXAM e.g. CBSE / ICSE / State Board], mixing several chapters so students cannot guess the topics. Total marks [TOTAL MARKS], time [TIME].

YOUR TASK:
1. Pull questions from across ALL the chapters I list, spread roughly evenly.
2. Deliberately shuffle the order so chapters are interleaved, not grouped.
3. Mix question types and difficulty: [e.g. 50 percent easy, 30 percent moderate, 20 percent hard].
4. Provide a complete answer key, and beside each answer note which chapter it came from.

INPUT: Chapters covered so far: [LIST CHAPTERS]. Use only these.

HOW TO WORK IT OUT: Solve every question internally first, then verify each final answer twice by independent methods. Confirm each listed chapter gets a fair share of marks. Never invent formulas or values. Keep mathematics legible in clean LaTeX or plain text. Confirm marks total exactly [TOTAL MARKS].

OUTPUT FORMAT, in this order:
1. HEADER: 'Surprise Revision Test - Mathematics', class, time, maximum marks, 'Compiled by Indrajeet Yadav'.
2. INSTRUCTIONS: 3 to 5 numbered lines.
3. THE TEST: numbered questions, interleaved chapters, marks beside each.
4. ANSWER KEY: final answer for every question, with the source chapter noted beside it.

RULES: Interleave chapters. Spread marks fairly across all listed chapters. Number every question. Marks total exactly.

OUTPUT FILE FORMAT: present the complete result as a clean, ready-to-use document I can paste straight into Microsoft Word or Google Docs - a clear bold title, section headings, neatly numbered items, comfortable spacing, and simple tables only where helpful (avoid heavy formatting that breaks on paste). It must also print cleanly to PDF on A4 (File > Print > Save as PDF). Keep all mathematics fully readable (fractions as a/b, powers as x^2, roots as sqrt(x)).

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.

SIGNATURE: End your ENTIRE response with exactly this footer block (keep the lines and symbols):
