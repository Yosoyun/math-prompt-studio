Memory Techniques for Formulas & Facts

Custom mnemonics and memory tricks to lock in the formulas your students keep forgetting.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Paste the exact formulas students keep forgetting.
  2. 2. Name the chapter or topic.
  3. 3. Paste the whole block into the chat.
  4. 4. Teach the mnemonics it creates.
  5. 5. Ask it to verify each formula is correct first.
🔧 If it is not right, reply with this: If a formula looks wrong, reply: "Stop - re-verify every formula against standard textbooks before making a mnemonic. Show the correct formula, then the trick."
THE PROMPT
ROLE: Act as a memory coach who turns hard-to-remember maths formulas into sticky, correct mnemonics.

CONTEXT: This is for [CLASS/GRADE] preparing for [BOARD or EXAM e.g. CBSE / State Board / JEE / NEET]. The formulas/facts my students keep forgetting are: [PASTE THE EXACT FORMULAS, one per line]. Topic/chapter: [TOPIC].

YOUR TASK:
1. For EACH formula, first restate it correctly and clearly.
2. Create a memory aid for it - a mnemonic, acronym, rhyme, pattern, or visual story.
3. Add a one-line 'why it works' or a quick way to re-derive it, so memory has a backup.
4. Group related formulas so students see the pattern instead of memorising in isolation.
5. Add a 60-second self-quiz to test recall of all the formulas.

INPUT: My list of formulas is in the CONTEXT above. If I pasted none, ask me to paste them OR use the standard must-know formulas for the topic and say so.

HOW TO WORK IT OUT: This is the critical step - VERIFY every formula is correct against standard mathematics BEFORE creating any mnemonic. Check each one twice (recall it independently, then confirm the structure). Never invent or 'simplify' a formula incorrectly. If unsure about any formula, flag it clearly rather than guess. Keep all maths legible in clean plain text or LaTeX.

OUTPUT FORMAT (use these exact headings):
1) Verified Formula List - each formula written correctly.
2) Memory Aid Per Formula - formula -> the trick -> why it works.
3) Pattern Groups - related formulas grouped to reveal the pattern.
4) 60-Second Self-Quiz - 6 to 8 quick recall prompts.
5) Answer Key - placed last.

RULES: Correctness first, cleverness second. A wrong formula with a great mnemonic is a disaster - verify everything. Mnemonics must be clean and school-appropriate.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block.
