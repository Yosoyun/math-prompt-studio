Sample / Model Paper in the Latest Exam Pattern

A model paper that mirrors the most recent official pattern, with section design and key.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Paste the prompt into a new chat.
  2. 2. Fill in class, board and total marks.
  3. 3. List the full syllabus to cover.
  4. 4. Send and review the pattern note at top.
  5. 5. Confirm the section design matches your board.
  6. For a trustworthy answer key: paste the finished paper into a FRESH chat and ask it to solve and grade it as an unseen exam (it catches its own slips).
🔧 If it is not right, reply with this: If the pattern looks outdated, reply: 'Restate the exact section structure (number of questions and marks per section) you assumed, then rebuild the paper to match the latest [BOARD] pattern.'
THE PROMPT
ROLE: Act as a mathematics paper-setter who specialises in latest-pattern model papers.

CONTEXT: I need a SAMPLE / MODEL paper for Class [CLASS/GRADE], [BOARD or EXAM e.g. CBSE / ICSE / State Board], that closely follows the most recent official examination pattern. Total marks [TOTAL MARKS], time [TIME e.g. 3 hours].

YOUR TASK:
1. First, state in 3 to 4 lines the exact pattern you are following: how many sections, how many questions per section, and marks per question.
2. Build the paper to match that structure precisely.
3. Include any modern elements that pattern uses (for example internal choice, case-study or assertion-reason), if applicable.
4. Provide a complete answer key.

INPUT: The full syllabus / chapters to cover is: [LIST CHAPTERS or TOPICS]. If I leave the pattern detail blank, assume the standard latest pattern for this class and clearly state your assumption at the top.

HOW TO WORK IT OUT: Solve every question internally before printing. Verify each final answer with two independent checks. Never invent formulas, values or theorems. Keep mathematics legible in clean LaTeX or clear plain text. Confirm all marks sum exactly to [TOTAL MARKS] before showing the paper.

OUTPUT FORMAT, in this order:
1. PATTERN NOTE: the section structure you are following.
2. HEADER BLOCK: 'Sample Paper - Mathematics', 'Class: [CLASS/GRADE]', 'Board: [BOARD or EXAM]', 'Time: [TIME]', 'Maximum Marks: [TOTAL MARKS]', 'Compiled by Indrajeet Yadav'.
3. GENERAL INSTRUCTIONS: numbered.
4. THE PAPER: sections labelled, marks beside every question, internal choices shown clearly with 'OR'.
5. ANSWER KEY: final answers plus short working outline for long questions.

RULES: Stay inside the listed syllabus. Make the structure faithful to the named board. Number every question. Marks total exactly.

OUTPUT FILE FORMAT: present the complete result as a clean, ready-to-use document I can paste straight into Microsoft Word or Google Docs - a clear bold title, section headings, neatly numbered items, comfortable spacing, and simple tables only where helpful (avoid heavy formatting that breaks on paste). It must also print cleanly to PDF on A4 (File > Print > Save as PDF). Keep all mathematics fully readable (fractions as a/b, powers as x^2, roots as sqrt(x)).

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.
