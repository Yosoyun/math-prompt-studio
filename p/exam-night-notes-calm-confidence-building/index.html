Exam-Night Notes (Calm, Confidence-Building)

Short, calming notes for the night before the exam.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Type the chapters and class.
  2. 2. Paste the prompt.
  3. 3. Read it in under 15 minutes.
  4. 4. Ask to add a calming tip if wanted.
🔧 If it is not right, reply with this: If it feels too heavy, reply: "Make this lighter and shorter - keep only the top 10 things, and verify each formula."
THE PROMPT
ROLE: Act as a calm, experienced mathematics teacher writing exam-night notes that steady nerves and lock in essentials.

CONTEXT: My [CLASS/GRADE] students sit their exam tomorrow on [CHAPTER NAME(S)] for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. Tonight is not for new learning - it is for a calm final pass. I want short, reassuring notes that cover only the surest mark-earners.

YOUR TASK:
1. List the 10 to 12 most important formulas and results, nothing more.
2. Give a 3-step plan for the most common question type per chapter.
3. Add a short 'do not panic' list of small mistakes to avoid in the hall.
4. Keep the tone calm and encouraging throughout.

INPUT: Here are the chapters: [CHAPTER NAME(S)]. If I paste the exam pattern, prioritise by it.

HOW TO WORK IT OUT: Choose only well-established, high-yield content. Test any quick result internally and VERIFY it two ways before listing. Never invent formulas or values. Keep all mathematics legible in clean LaTeX or clear plain text. If something is risky to revise the night before, leave it out and say so in one line.

OUTPUT FORMAT (in this order):
1. Title: [CHAPTER NAME(S)] - Exam-Night Notes
2. The 10-12 Essentials (each one line)
3. Question-Type Game Plans (3 steps each)
4. In-Hall Do-Not Mistakes (6 short bullets)
5. One Calm Closing Line

RULES: Short and reassuring. No new or obscure material. Everything standard and exam-correct. Readable in under 15 minutes. No fear-inducing language. State one assumption line if scope is unclear.

OUTPUT FILE FORMAT: present the complete result as a clean, ready-to-use document I can paste straight into Microsoft Word or Google Docs - a clear bold title, section headings, neatly numbered items, comfortable spacing, and simple tables only where helpful (avoid heavy formatting that breaks on paste). It must also print cleanly to PDF on A4 (File > Print > Save as PDF). Keep all mathematics fully readable (fractions as a/b, powers as x^2, roots as sqrt(x)).

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.

SIGNATURE: End your ENTIRE response with exactly this footer block (keep the lines and symbols):
