Worksheet With Full Answer Key (Self-Marking Ready)

A complete worksheet plus a detailed answer key with marking notes for self or peer marking.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Name the topic and question count.
  2. 2. Add class and board.
  3. 3. Send the prompt.
  4. 4. Verify the marking notes.
  5. 5. Print sheet plus key.
🔧 If it is not right, reply with this: If the key lacks working, reply: "Add short step-by-step working for every answer in the key, and show where each mark is awarded."
THE PROMPT
ROLE: Act as a mathematics teacher who produces worksheets with detailed, marking-ready answer keys.

CONTEXT: I teach [CLASS/GRADE] under [BOARD or EXAM e.g. CBSE / ICSE / State Board]. I want a worksheet on [TOPIC] with an answer key detailed enough that students can self-mark or peer-mark accurately, knowing exactly where each mark is earned.

YOUR TASK:
1. Create [NUMBER, e.g. 12] questions on [TOPIC], ordered easy to hard, with marks per question and a total.
2. For the answer key, give for EACH question: the final answer, the key working steps, and a 'mark breakdown' showing how the marks split (for example, '1 mark for correct setup, 1 mark for correct simplification, 1 mark for final answer').
3. Add a short 'accept also' note where more than one correct form is valid (for example, decimal or fraction).
4. Keep questions unambiguous so marking is objective.

INPUT: Here is the topic, number of questions, and any marking preferences: [PASTE DETAILS HERE].

HOW TO WORK IT OUT: Solve every question internally first. VERIFY each final answer two independent ways. Make the mark breakdowns add up to each question's marks. Never invent formulas or values. Keep all maths fully legible in clean LaTeX or clear plain text.

OUTPUT FORMAT (exact headings, in order):
- TITLE BLOCK: 'Worksheet with Answer Key', Topic, Class, Total Marks, and 'Compiled by Indrajeet Yadav'.
- WORKSHEET: Q1 to Q[NUMBER] with marks.
- DETAILED ANSWER KEY: for each question, final answer + working + mark breakdown + 'accept also' note where relevant.
- MARKING TIPS: 3 short bullets to help a student peer-mark fairly.

RULES: Questions unambiguous and objectively markable. Mark breakdowns must total correctly. Key must match worksheet. Simple, clear wording.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: 'Prepared by Indrajeet Yadav', and put 'Compiled by Indrajeet Yadav' in the title block.
