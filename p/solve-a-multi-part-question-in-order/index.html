Solve a Multi-Part Question in Order

Each part (a), (b), (c) solved in order, with earlier answers carried into later parts.

Attach a photo of the question first  ·  Best tool:  ChatGPT or Gemini (image)

✅ How to use this effectively

  1. 1. Photograph the whole question with all sub-parts.
  2. 2. Attach the photo and paste this prompt.
  3. 3. Send and read parts in order.
  4. 4. Check each part uses the earlier result.
  5. 5. Ask for any skipped sub-part.
🔧 If it is not right, reply with this: If it jumbles the parts, reply: "Solve strictly in order a, b, c... and use the answer from each part as input to the next where required."
THE PROMPT
ROLE: Act as a mathematics teacher who solves a multi-part question carefully, in order, carrying results forward correctly.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET], topic [TOPIC]. The photo shows ONE question with several connected parts, such as (a), (b), (c). Later parts often depend on earlier answers.

YOUR TASK:
1. Read the question in the attached photo and restate it, listing every sub-part in order.
2. Solve the parts strictly in the given order: (a), then (b), then (c), and so on.
3. Where a later part needs an earlier result, use it explicitly and say so.
4. Award-style: note the likely marks for each part if marks are shown.

INPUT: I have attached a photo of the multi-part problem. If any sub-part or symbol is unclear, tell me which one, state your best assumption, and continue.

HOW TO WORK IT OUT: Solve each part silently first, in order. VERIFY each part's answer with at least one quick check, and verify the FINAL part with TWO independent checks. If an earlier part is wrong, every later part using it is also affected - so be especially careful with the early parts. Never invent formulas or values; name standard results. Keep all mathematics fully legible in clean LaTeX or clear plain text, one step per line.

OUTPUT FORMAT: First a line "Parts found: (a), (b), ...". Then for EACH part use this block exactly:
--- Part (letter) ---
- Asks: restated sub-question.
- Uses from earlier: which previous result it depends on, or "None".
- Solution: numbered steps, one idea per line, short reasons.
- Answer: clearly stated (with marks if shown).
After all parts, add: "Final verification" with Check 1 and Check 2 on the last part.

RULES: Never solve parts out of order. Always state when you reuse an earlier answer. Show units where needed. Use plain ASCII only with straight quotes. Be exact unless an estimate is requested.


MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.
At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block at the top.
