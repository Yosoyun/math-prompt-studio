Rank 3 Student Answers (best to weakest)

Three answers ranked with marks and reasons, so grading is consistent and defensible.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. Paste answers labelled A, B and C.
  2. Type the question and total marks.
  3. Set the class and board or exam.
  4. Press Enter for the ranking and scores.
  5. Ask for a re-rank if you disagree.
🔧 If it is not right, reply with this: If a ranking seems off, reply: "Answer C earns more method marks than you gave; re-score all three on the same rubric and re-rank."
THE PROMPT
ROLE: Act as a moderation examiner who ranks multiple student answers fairly on one rubric.

CONTEXT: These are [CLASS/GRADE] answers for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE]. Topic: [TOPIC]. The question is worth [TOTAL MARKS]. I want all three judged by the SAME standard so my marking is consistent.

YOUR TASK:
1. Solve the question yourself and fix a single step-wise mark scheme out of [TOTAL MARKS].
2. Apply that identical mark scheme to Answer A, Answer B and Answer C.
3. Rank them best to weakest with scores and clear reasons.
4. Note what separates first place from last.

INPUT: The question is: [PASTE THE QUESTION HERE]. Answer A: [PASTE A]. Answer B: [PASTE B]. Answer C: [PASTE C]. Total marks: [TOTAL MARKS].

HOW TO WORK IT OUT: Solve internally first, then VERIFY your model answer with two independent checks and state them. Score every answer against the same rubric rows; award method marks even when the final number is wrong. Be even-handed; do not reward neat handwriting over correct reasoning. Never invent formulas or values. Keep maths legible.

OUTPUT FORMAT (exact headings):
1. Shared mark scheme - the rubric rows that sum to [TOTAL MARKS].
2. Scorecard - a table: Answer | marks per row | total | one-line reason.
3. Ranking - 1st, 2nd, 3rd with the deciding factor for each gap.
4. Common strength and common weakness across the three.
5. Model answer and verification - the full-marks version with two checks.

RULES: Plain ASCII only, straight quotes. Use the exact same rubric for all three; the totals must match the rows. Justify every placement.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block.
