Fill-in-the-Blank Quiz

A fill-in-the-blank worksheet that targets key terms and formula steps, with a clean answer key.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Enter class, board and chapter.
  2. 2. Say how many fill-in items you want.
  3. 3. Send the prompt.
  4. 4. Print Section 1 for students.
  5. 5. Keep Section 2 as your key.
🔧 If it is not right, reply with this: If a blank has more than one correct word, reply: 'Blank 6 accepts two answers - reword the sentence so only one word fits, then re-verify.'
THE PROMPT
ROLE: Act as a mathematics teacher who writes fill-in-the-blank revision quizzes.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM e.g. CBSE / ICSE / State Board]. The chapter is [CHAPTER NAME]. I want [NUMBER, e.g. 15] fill-in-the-blank items.

YOUR TASK:
1. Write [NUMBER] sentences each with exactly ONE blank shown as ______ .
2. Mix three types: (a) key term or definition blanks, (b) formula blanks (a missing symbol or term in a formula), (c) value blanks (a missing number from a short result).
3. Each blank must have a single, unambiguous correct answer.
4. Order from easy to hard.

INPUT: Here is the sub-topic or specific terms to test: [PASTE TOPIC OR LEAVE BLANK].

HOW TO WORK IT OUT: Fill each blank yourself and VERIFY it by checking the sentence reads correctly with only that answer, and by a second check for any value blank (recompute it). Do not invent terms or values. Keep all maths legible.

OUTPUT FORMAT:
SECTION 1 - QUIZ: The numbered sentences with blanks, no answers.
SECTION 2 - ANSWER KEY: Numbered list of the correct word, symbol or value for each blank, with a 3-4 word note for value blanks showing the quick check.
SECTION 3 - WORD BANK (optional): A shuffled list of all answers, for an easier version.

RULES: One blank per sentence. Only one correct answer per blank. No trick phrasing. Difficulty suited to the class above.


MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.
At the very end add a footer line: 'Prepared by Indrajeet Yadav', and put 'Compiled by Indrajeet Yadav' in the title block at the top.
