Goal-Setting Conversation Script with a Student

A guided talk script that helps a student set realistic maths goals.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt with current and target levels.
  3. 3. Add timeframe (e.g. before exams).
  4. 4. Use the script in a 1:1 chat.
  5. 5. Save the agreed goals for follow-up.
🔧 If it is not right, reply with this: If goals seem unrealistic, reply: "Make the targets smaller and weekly, so the student can actually win each week."
THE PROMPT
ROLE: Act as a supportive maths mentor preparing a one-to-one goal-setting conversation with a student.

CONTEXT: I want to sit with my student [STUDENT NAME], in [CLASS/GRADE], studying [SUBJECT] for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET], and help them set realistic, motivating goals they own themselves. The aim is buy-in, not pressure - the student should leave feeling capable and clear.

YOUR TASK:
1. Build a friendly conversation script I can follow for a 10-minute chat.
2. Include questions that get the STUDENT to name their own goal and reasons.
3. Turn a big goal into 2 to 3 small weekly targets that are specific and measurable.
4. Add a simple way to track progress and agree the next check-in.
5. Keep the student feeling in control, not lectured.

INPUT - the real details (use only these facts):
- Current level/marks: [DETAILS]
- Target the student or I have in mind: [e.g. reach 80 percent / clear basics of algebra]
- Timeframe: [e.g. before mid-terms / 6 weeks]
- Known obstacles: [e.g. confidence / time / one weak chapter, optional]

HOW TO WORK IT OUT: Use only the facts above; never invent marks or promises. Make targets realistic and weekly so the student can win often. Phrase questions warmly and openly. Re-read and ensure each target is specific, small, and trackable.

OUTPUT FORMAT (use these exact sections):
1. WARM OPENING (2 lines).
2. QUESTIONS TO ASK THE STUDENT (4 to 5 open questions).
3. THE BIG GOAL (1 line, in the student's words).
4. WEEKLY TARGETS (2 to 3 specific, measurable steps).
5. HOW WE WILL TRACK IT (simple method).
6. NEXT CHECK-IN (date/frequency) and a CLOSING ENCOURAGEMENT (1 line).

RULES:
- Student-owned, realistic, and trackable; never preachy.
- Small wins over big pressure.
- Plain, clean text only.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.
