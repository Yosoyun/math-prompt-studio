Explain This Concept to a Parent

A warm, jargon-free note that helps a parent support their child.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt.
  3. 3. Add concept and class level.
  4. 4. Send and read the parent note.
  5. 5. Send it home or read at a meeting.
🔧 If it is not right, reply with this: If it sounds too school-like, reply: "Make it warmer and assume the parent has not studied maths since school. Add one thing they can do at home."
THE PROMPT
ROLE: Act as a kind mathematics teacher writing to a parent who has not done maths in many years.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. Parents want to help at home but do not understand this concept: [WRITE THE CONCEPT HERE]. I want a short note that explains it simply and tells them how to support their child.

YOUR TASK:
1. Explain what the concept is and why the child is learning it, in everyday language.
2. Assume the parent has forgotten almost all school maths.
3. Give one simple everyday example a parent will recognise.
4. Suggest two or three small things the parent can do at home to help.
5. Reassure the parent that they do NOT need to be good at maths to support their child.

INPUT: Here is the concept: [PASTE OR TYPE IT HERE]. Class level: [CLASS/GRADE].

HOW TO WORK IT OUT: Keep the maths in plain words; if any number appears, solve and VERIFY it two ways before using it so the parent is never told something wrong. Never invent formulas or scary jargon. Keep everything legible and gentle.

OUTPUT FORMAT (a warm note with these exact headings):
- DEAR PARENT (a friendly 1 to 2 line opening)
- WHAT YOUR CHILD IS LEARNING (plain explanation)
- AN EXAMPLE FROM DAILY LIFE
- HOW YOU CAN HELP AT HOME (2 to 3 simple actions)
- A WORD OF REASSURANCE (1 to 2 lines)

RULES: No jargon, no symbols, no homework-policing tone. Warm and respectful throughout. The help suggestions must need no maths skill from the parent.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block.
