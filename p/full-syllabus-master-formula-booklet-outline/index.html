Full-Syllabus Master Formula Booklet Outline

A structured chapter-by-chapter outline for a complete formula booklet.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Paste the prompt into the chat.
  2. 2. Enter the class, board, and full chapter list.
  3. 3. Send and review the booklet outline.
  4. 4. Ask it to draft one chapter in full.
  5. 5. Assemble chapters into one document.
🔧 If it is not right, reply with this: If a chapter is missing or thin, reply: "Chapter [name] is missing key sections. Re-outline it with all standard sub-topics and verify completeness."
THE PROMPT
ROLE: Act as a senior mathematics teacher planning a full-year master formula booklet.

CONTEXT: I teach [CLASS/GRADE] for the [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. I want to build a complete MASTER FORMULA BOOKLET for the whole syllabus. First I need a clear, well-structured OUTLINE I can then fill chapter by chapter.

YOUR TASK:
1. Produce a chapter-by-chapter outline of the full formula booklet for this class and board.
2. For each chapter, list the sub-headings under which formulas will go (for example: Definitions, Identities, Standard Results, Special Cases).
3. Estimate how many formulas roughly belong in each chapter.
4. Suggest a consistent layout and notation to use across all chapters.
5. Recommend an order and a sensible page budget so the booklet stays usable.

INPUT: Here is the class, board, and chapter list. Class/Board: [PASTE HERE]. Chapters (if known): [PASTE FULL CHAPTER LIST HERE]. If I do not paste chapters, use the standard full syllabus for this class and board.

HOW TO WORK IT OUT: Internally recall the full standard syllabus and confirm no chapter or major sub-topic is missing. VERIFY two ways: (a) compare your chapter list against the standard syllabus table of contents; (b) re-scan each chapter's sub-headings for any obvious gap. Do not yet write out the formulas - this is an outline - but everything you list must be standard and real. Never invent chapters or topics. Keep maths notation guidance legible.

OUTPUT FORMAT (use this exact order):
1. TITLE: 'Master Formula Booklet - Outline (Class [X], [Board])'.
2. 'Compiled by Indrajeet Yadav'.
3. A numbered chapter list; under each chapter, the sub-headings and an approximate formula count.
4. A 'House style' section: notation rules, layout, page budget.
5. A 'Build order' suggestion for filling it in batches.
6. Footer line.

RULES: Complete and standard coverage. No invented topics. Clear, consistent structure a teacher can hand to a typist.

OUTPUT FILE FORMAT: present the complete result as a clean, ready-to-use document I can paste straight into Microsoft Word or Google Docs - a clear bold title, section headings, neatly numbered items, comfortable spacing, and simple tables only where helpful (avoid heavy formatting that breaks on paste). It must also print cleanly to PDF on A4 (File > Print > Save as PDF). Keep all mathematics fully readable (fractions as a/b, powers as x^2, roots as sqrt(x)).

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.
