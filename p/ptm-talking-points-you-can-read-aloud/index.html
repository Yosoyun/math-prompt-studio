PTM Talking Points You Can Read Aloud

A tidy one-page script of points to say at a parent-teacher meeting.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt and add real notes.
  3. 3. Print or open the script on your phone.
  4. 4. Read it during the meeting.
  5. 5. Ask for a Hindi version if helpful.
🔧 If it is not right, reply with this: If a point is wrong, reply: "Point 3 is not accurate - here is the correct fact: [...]. Rewrite that point only."
THE PROMPT
ROLE: Act as a calm, professional maths teacher preparing for a parent-teacher meeting (PTM).

CONTEXT: I will meet the parent of [STUDENT NAME], in [CLASS/GRADE], studying [SUBJECT] for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. I have only a few minutes per parent and want clear talking points so the conversation stays positive, specific, and useful.

YOUR TASK:
1. Turn my rough notes into a clean PTM script I can read aloud or glance at.
2. Start positive, then raise concerns gently, then agree next steps together.
3. Give me 3 short questions to ASK the parent (about home study, sleep, stress, support).
4. Keep it timed for about 4 to 5 minutes of talking.

INPUT - my rough notes (use only these facts):
- Strengths: [LIST]
- Concerns: [LIST]
- Recent marks or work: [DETAILS, optional]
- Anything sensitive to handle carefully: [e.g. exam anxiety / family situation, optional]

HOW TO WORK IT OUT: Use only the facts I provide; never invent marks or incidents. Where a fact is missing, insert a clearly marked [placeholder]. Keep language warm and free of jargon. Re-read and soften any wording that could make a parent defensive.

OUTPUT FORMAT (use these exact sections):
1. OPENING (warm, 2 lines).
2. STRENGTHS TO HIGHLIGHT (3 bullet points).
3. CONCERNS TO RAISE GENTLY (2 to 3 bullet points, each with a kind phrasing).
4. QUESTIONS TO ASK THE PARENT (3 bullets).
5. AGREED NEXT STEPS (2 to 3 bullets, simple and doable).
6. CLOSING LINE (1 encouraging sentence).

RULES:
- Specific, honest, and supportive; never blaming the child or parent.
- Short bullets I can scan quickly.
- Plain, clean text only.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.
