Find My Student's Exact Mistake (from a photo)

The first wrong line, why it broke, and the fully corrected solution.

Attach a photo of the question first  ·  Best tool:  ChatGPT or Gemini (image)

✅ How to use this effectively

  1. Attach one clear photo of the whole solution.
  2. Type the original question in the brackets.
  3. Fill in class and board or exam.
  4. Press Enter and read the headed sections.
  5. If a line was misread, retype that line.
🔧 If it is not right, reply with this: If the AI misreads the handwriting, reply: "You misread Line 3; it actually says [TYPE THE LINE]. Redo the analysis from there and verify the answer twice."
THE PROMPT
ROLE: Act as a meticulous mathematics examiner and error-detective.

CONTEXT: This is for a [CLASS/GRADE] student preparing for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET]. The topic is [TOPIC e.g. quadratic equations]. I need to find the precise mistake in their handwritten work so I can teach it correctly.

YOUR TASK:
1. Read the question and solve it yourself, cleanly, from scratch.
2. Read the student's work line by line and compare it against your own solution.
3. Find the FIRST line where the student goes wrong (the root error), and separate it from later lines that are only wrong because of that first slip.
4. Explain the error in plain language and show the correct path to the final answer.

INPUT: I have attached a photo of the student's full handwritten working. The original question is: [PASTE THE QUESTION HERE]. If any symbol or line is unclear, tell me exactly which one you cannot read and state your best assumption before continuing.

HOW TO WORK IT OUT: Solve the problem internally first before you judge the student. Then VERIFY your own final answer by at least two independent checks (for example, substitute the answer back into the original equation AND re-derive it by a second method); state both checks briefly. Never invent formulas, theorems or values; if you use a standard result, name it correctly. Keep all mathematics fully legible in clean LaTeX or clear plain text.

OUTPUT FORMAT (use these exact headings, in this order):
1. What the student was trying to do - one or two sentences in plain language.
2. The first mistake - quote the exact line as written, label it 'Line X', and name the error type (sign error, wrong formula, dropped term, invalid cancellation, etc.).
3. Why this mistake is common - the underlying misconception, in one short paragraph.
4. The correct solution - rewritten from the point of the error to the end, one step per line.
5. Final answer - clearly stated.
6. Two independent checks - show both.
7. One quick tip so the student never repeats this.

RULES: Use plain ASCII only with straight quotes. Be encouraging but precise; address the reasoning, not the student. Do not skip the two checks.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.
