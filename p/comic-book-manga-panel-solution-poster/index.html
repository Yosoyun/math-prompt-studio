Comic-Book / Manga Panel Solution Poster

A dynamic ink-and-halftone comic page where the maths solution unfolds across bold action panels with speech-burst captions.

Makes images - needs an image AI  ·  Best tool:  ChatGPT Plus (image) or Gemini

✅ How to use this effectively

  1. Paste the full problem text into the [PASTE] slot, or attach a clear, well-lit photo
  2. Tell it the exact answer if you already know it, so it can cross-check
  3. Ask it to confirm the verified answer in text BEFORE drawing
  4. Request high resolution / best quality vertical 4:5
  5. Ask for one more render if any digit or symbol looks malformed
🔧 If it is not right, reply with this: If the maths is wrong, reply: 'Re-solve step by step, state the verified final answer in plain text first, then redraw.' If cluttered: 'Fewer panels, bigger lettering, move all ink splashes to the margins, nothing over the equations.'
THE PROMPT
Create a premium vertical educational mathematics poster in a bold comic-book / manga panel style, approximately 4:5 portrait.

INPUT: Read the maths problem I give you (pasted here or attached as a photo). SOLVE it yourself and verify the arithmetic by an independent check BEFORE rendering. Then put the VERIFIED solution into the poster. Problem: [PASTE THE COMPLETE PROBLEM HERE, or attach a clear photo].

FORMAT & COMPOSITION: A single inked comic page. Crisp black ink linework, dramatic Ben-Day halftone dots, bold primary palette (electric blue, hero red, sunburst yellow) on clean off-white paper. Thick black panel gutters divide the page into a clear top-to-bottom reading sequence of 4 to 5 rectangular panels. Action lines, motion streaks, small star-burst pops, and a torn-paper energy splash ONLY in the outer margins and corners. Generous white gutters; balanced negative space; nothing decorative ever crosses the mathematics.

TYPOGRAPHY: Bold hand-inked comic lettering for headings and captions; clean upright sans for the actual maths so every symbol is razor sharp. Render notation exactly: fractions as clean horizontal bars, factorials n!, permutations nPr (superscript P), combinations nCr (binomial), summation, subscripts, superscripts. Never distort numbers, symbols, or punctuation.

CONTENT (strict top-to-bottom hierarchy): (1) A bold title banner across the top: '[Q-number] - [short topic title]'. (2) A 'Problem:' caption box containing the question VERBATIM. (3) A 'Method: [precise method name]' heading in a punchy caption strip. (4) The solution as 3 to 6 concise steps, one idea per panel, in clean speech-style boxes; where the problem type calls for it include a neat central SLOT/POSITION diagram (e.g. F _ _ R with little braces labelled 'fixed') OR a small tidy rank/ledger TABLE drawn in comic-ink style. (5) A bottom hero panel with a thick double-outlined burst box: 'Final Answer: [value]'.

SIGNATURE: a small elegant handwritten signature 'Indrajeet Yadav' in a bottom corner, tasteful, not overlapping any mathematics.

ACCURACY SAFEGUARDS: copy every word and symbol exactly; verify the arithmetic independently before rendering; never drop restrictions like 'without repetition', 'distinct', 'identical', 'fixed', 'do not come together'; show ONE method only (no alternative method); use correct permutation/combination/factorial notation; keep it readable at mobile-phone size; maintain title -> problem -> method -> reasoning -> answer hierarchy; NO malformed numerals, NO duplicated or garbled digits, NO spelling mistakes, NO pseudo-handwriting gibberish, NO random extra equations, NO decoration over the text. The maths must be perfectly legible and 100 percent correct.
