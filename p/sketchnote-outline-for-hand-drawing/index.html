Sketchnote Outline (for Hand-Drawing)

A sketchnote plan you draw by hand: layout, icons to draw, and labels.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Type the chapter and class.
  3. 3. Paste the prompt below.
  4. 4. Send and read the layout plan.
  5. 5. Draw it on paper or a board.
🔧 If it is not right, reply with this: If an icon idea is too hard to draw, reply: "Suggest a simpler stick-figure icon for that node that any teacher can draw." If content is thin, reply: "Add two more nodes with key formulas."
THE PROMPT
ROLE: Act as a mathematics teacher and a sketchnoting coach who plans hand-drawn visual notes for the blackboard or paper.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE]. I cannot draw fancy graphics on the computer, but I CAN draw on paper or the board. Give me a clear SKETCHNOTE OUTLINE (text instructions) telling me exactly where to place things and which simple icons to draw. No image needed - this is a drawing plan for me.

YOUR TASK:
1. Describe a single-page sketchnote layout for the chapter I name.
2. Specify zones by position (top-centre, left column, right column, bottom band).
3. For each node give: the label text, the key formula or idea, and a SIMPLE icon I can hand-draw (described in words, like "a small triangle" or "a stick clock").
4. Suggest where to draw arrows connecting related nodes.
5. Keep all icons drawable by a non-artist.

INPUT: Here is the chapter and the points to include: [PASTE CHAPTER AND KEY POINTS HERE]. If only the chapter name is given, use standard [BOARD] content for [CLASS/GRADE].

HOW TO WORK IT OUT: Plan the whole page first. VERIFY every formula and idea twice before listing it (syllabus check plus a quick sanity check). Never invent formulas. Keep icons trivially simple to draw. Write all maths in clean plain text.

OUTPUT FORMAT (exactly this):
SKETCHNOTE PLAN: <chapter>
TOP-CENTRE (title bubble): <text>, icon to draw: <simple icon>
LEFT COLUMN:
   Node 1 - label: <text> | formula/idea: <...> | icon: <simple icon>
   Node 2 - ...
RIGHT COLUMN:
   Node 3 - ...
BOTTOM BAND (key formulas strip): <formulas>
ARROWS TO DRAW: connect <node> to <node> because <reason>
DRAWING TIPS: 3 quick tips for keeping it neat on the board.

RULES: Every icon must be drawable in 5 seconds by hand. Be specific about positions. Match depth to [CLASS/GRADE].

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block.
