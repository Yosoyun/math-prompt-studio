THE PROMPT
ROLE: Act as a maths teacher who draws crisp number lines for inequalities and intervals.

CONTEXT: I teach [CLASS/GRADE] under [BOARD or EXAM e.g. CBSE / ICSE / JEE]. Students confuse open and closed circles and shading direction, so the figure must be unmistakable.

YOUR TASK:
1. Draw a horizontal number line with evenly spaced integer marks covering the relevant range plus a little extra on each side.
2. Place the solution on the line: use an OPEN circle for strict inequalities (< or >) and a FILLED circle for inclusive ones (<= or >=).
3. Shade or thicken the part of the line that is the solution, with arrows if it goes to infinity.
4. Write the interval notation and the inequality above the shaded region.
5. Label every critical value on the line.

INPUT: Here is the inequality (or solution set): [PASTE IT HERE, e.g. -2 <= x < 5, or "x > 3 or x <= -1"].

HOW TO WORK IT OUT: Solve the inequality fully on the side first if it is not already simplified. Then VERIFY two ways: pick one test value inside the claimed solution and confirm it satisfies the inequality, and pick one value outside and confirm it fails. State both test results. Never mix up open and closed circles; never shade the wrong side. If the solution is empty or all real numbers, say so plainly.

OUTPUT FORMAT:
1. SOLVED FORM: the simplified inequality and its interval notation.
2. THE NUMBER LINE: one clean image with correct circles, shading and labels.
3. TWO TESTS: one inside value and one outside value, each with a yes/no.

RULES: Open versus filled circles must match the inequality signs exactly. Arrows only where the solution is unbounded. Plain white background, large readable numbers.
