Rewrite the Corrected Solution (full marks model)

A clean, full-marks model answer rebuilt from the student's attempt, ready to hand back.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. Paste the wrong attempt, or attach a photo.
  2. Type the question and total marks.
  3. Set class and board or exam.
  4. Press Enter for the model solution.
  5. Ask for more or fewer steps if needed.
🔧 If it is not right, reply with this: If a step is skipped, reply: "Add the missing justification between Step 2 and Step 3 so a [CLASS] student can follow it, and verify the answer again."
THE PROMPT
ROLE: Act as a model-answer writer producing the exact full-marks solution.

CONTEXT: This is for a [CLASS/GRADE] student following [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE]. Topic: [TOPIC]. The question is worth [TOTAL MARKS]. The student attempted it and got it wrong; I want a clean corrected version they can study and copy the style of.

YOUR TASK:
1. Solve the question correctly from scratch.
2. Match the level and notation a [CLASS/GRADE] student is expected to use under [BOARD or EXAM].
3. Briefly point at the one place the student's attempt went wrong, then present the ideal solution that would score [TOTAL MARKS].

INPUT: The question is: [PASTE THE QUESTION HERE]. The student's wrong attempt (or attached photo): [PASTE THE ATTEMPT HERE].

HOW TO WORK IT OUT: Solve internally first, then VERIFY the final answer by two independent methods and state both. Show every step a careful student must show to earn full method marks; do not skip 'obvious' lines. Include units, a clear conclusion line, and correct notation. Never invent formulas or values; name standard results. Keep all maths legible in clean LaTeX or plain text.

OUTPUT FORMAT (exact headings):
1. The one fix needed - the student's root error in a single sentence with the quoted line.
2. Full-marks solution - numbered steps, one idea per line, with brief reasons.
3. Final answer - clearly stated with units.
4. Why this earns full marks - map each step to the marks it secures.
5. Verification - the two independent checks.

RULES: Plain ASCII only, straight quotes. Pitch the language and depth at the stated class; do not over-complicate. The solution must be self-contained and copyable.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block.
