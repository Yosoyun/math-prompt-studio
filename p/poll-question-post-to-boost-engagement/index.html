Poll Question Post to Boost Engagement

Five poll questions with options and the correct answer revealed.

Works on any free AI  ·  Best tool:  Any AI chat (ChatGPT/Gemini/Claude/Copilot)

✅ How to use this effectively

  1. 1. Open any free AI chat.
  2. 2. Paste the prompt below.
  3. 3. Add your topic.
  4. 4. Send and read the five polls.
  5. 5. Add one to your story or channel.
🔧 If it is not right, reply with this: If too many options are obviously wrong, reply: "Make the options closer in difficulty so the poll is a real challenge, and re-verify the answer."
THE PROMPT
ROLE: Act as a maths quiz and poll designer who creates polls that get high participation and start friendly debate.

CONTEXT: I post polls (Instagram story polls, Telegram polls, or LinkedIn polls) for [CLASS/GRADE] students preparing for [BOARD or EXAM e.g. JEE / NEET / CBSE]. Good polls have 2 to 4 tempting options and a satisfying reveal.

YOUR TASK:
1. Create FIVE poll questions on the topic I name.
2. Each poll has a short question and 2 to 4 options.
3. Mix poll types: some with one correct answer, and one or two opinion polls (for example, 'which method do you prefer?').
4. For factual polls, mark the correct option and give a one-line reason.
5. Add a follow-up comment line to keep the conversation going.

INPUT: My poll topic is: [PASTE TOPIC, e.g. quadratic equations / which is harder, integration or differentiation].

HOW TO WORK IT OUT: For every factual poll, solve it internally and VERIFY the correct option by a second independent check. Confirm only one option is correct in factual polls. Never invent values. For opinion polls, do not mark a 'correct' answer. Keep maths in clean plain text (^ for powers, sqrt() for roots, / for fractions).

OUTPUT FORMAT: For each of the 5 polls use this structure.
POLL [number] - Type: (factual or opinion)
Question: ...
Options: A) ... B) ... (C) ... D) ... as needed
Correct option (factual only): letter + one-line reason
Verification (factual only): one line showing the check
Follow-up comment line: ...
At the end add: ACCURACY CHECK - one line confirming all factual polls were verified.

RULES: Exactly one correct option in factual polls. No marked answer in opinion polls. Fun, inviting tone. Plain ASCII, straight quotes only.

MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.

SIGNATURE: At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the header block.
