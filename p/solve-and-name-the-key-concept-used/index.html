Solve and Name the Key Concept Used

A full solution plus a clear statement of the main concept and why it applies here.

Attach a photo of the question first  ·  Best tool:  ChatGPT or Gemini (image)

✅ How to use this effectively

  1. 1. Photograph the question clearly.
  2. 2. Attach the photo to the chat.
  3. 3. Paste this prompt and send.
  4. 4. Read the named concept and the solution.
  5. 5. Ask for two similar practice questions if useful.
🔧 If it is not right, reply with this: If the concept seems off, reply: "Re-examine the question type and name the precise concept or theorem it tests, then justify why it applies."
THE PROMPT
ROLE: Act as a mathematics teacher who solves a problem and clearly names the core concept it is testing.

CONTEXT: I teach [CLASS/GRADE] for [BOARD or EXAM e.g. CBSE / ICSE / State Board / JEE / NEET], topic [TOPIC]. I want my students to recognise WHAT KIND of problem this is, not just get the answer.

YOUR TASK:
1. Read the question in the attached photo and restate it in clean text.
2. Identify the single main concept, theorem, or technique the question is built on.
3. Explain in one short paragraph why that concept is the right tool for this question.
4. Then solve the problem fully using that concept.

INPUT: I have attached a photo of the problem. If any part is unclear, tell me which symbol or line you cannot read, state your best assumption, and continue.

HOW TO WORK IT OUT: Solve the problem silently first. Then VERIFY the final answer with TWO independent checks (for example substitute back, and re-derive a second way). If the checks disagree, fix the error before replying. Never invent formulas, theorems, or values; name any standard result correctly and precisely. Keep all mathematics fully legible in clean LaTeX or clear plain text, one step per line.

OUTPUT FORMAT (use these exact headings):
1. Question as I read it.
2. Key concept tested - name it in bold-style capitals, e.g. KEY CONCEPT: ..., in one line.
3. Why this concept applies - one short paragraph.
4. Step-by-step solution - numbered, one idea per line, short reasons.
5. Final answer - clearly stated.
6. Verification - Check 1 and Check 2, each in one or two lines.
7. How to spot this type next time - one or two recognition cues.

RULES: Name exactly one primary concept; if a supporting concept is needed, list it as secondary. Do not give a vague concept name - be specific (for example "Pythagoras theorem" not "geometry"). Use plain ASCII only with straight quotes. Be exact unless an estimate is requested.


MATHS FORMATTING: Write ALL mathematics so it reads correctly in a normal chat and pastes cleanly into Word or Google Docs. Use plain readable notation - for example a/b for fractions, x^2 for powers, sqrt(x) for roots, and symbols like <=, >=, !=, +-, pi, theta, integral and sum. Do NOT output raw LaTeX or code such as \frac{}{}, \sqrt{}, x^{2}, $...$ or \( ... \) - these appear as broken symbols to a teacher in a free chat.
At the very end add a footer line: "Prepared by Indrajeet Yadav", and put "Compiled by Indrajeet Yadav" in the title or header block at the top.

CONTEXT CHECK: Before you begin, if any [SQUARE-BRACKET] above is still empty, or I have not given you the question / photo / data you need, ask me for exactly those and wait - never guess, and never answer with the placeholder text still in brackets.

GROUND RULES (always follow): Do not invent statistics, percentages, marks distributions you were not given, exam cut-offs, solve-time figures, NCERT or textbook page/line numbers, citations, or historical exam data. If you are unsure of any fact or number, say so and clearly label it an estimate; keep established fact separate from your own inference. Every step must be mathematically correct and at the stated class/board/exam level.
